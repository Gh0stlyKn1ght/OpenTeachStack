## The Teacher Problem

Teachers often use words like course, website, app, LMS, and knowledge base as if they mean the same thing.

They do not.

When those containers blur, the teacher may publish a page and think the course is done, or build an app and forget the lesson sequence.

## The Idea

Each container has a job.

| Container | Plain meaning | Teacher risk |
|---|---|---|
| Course | The learning path, goals, sequence, evidence, and review routines | Calling a pile of pages a course |
| Website | A published place people can visit | Publishing without checking learning flow |
| App | An interactive tool that does something | Building features before naming the classroom need |
| Knowledge base | A reference library for help and reuse | Confusing support material with instruction |
| LMS | A classroom platform for assignments and delivery | Letting the platform decide the course structure |

## Classroom Scenario

A teacher creates a beautiful resource site with links, videos, and downloads. Students still ask, "What do I do first? What do I submit? How is this graded?"

The website exists. The course path does not.

## Weak Version

```text
I published the site, so the course is ready.
```

Publishing is not the same as teaching.

## Better Version

```text
The site holds the course content. The course still needs targets, sequence, student tasks, evidence, safety checks, and revision notes.
```

This separates the container from the instructional system.

## Build Step

Add these four terms to your vocabulary map:

- course
- website
- app
- knowledge base

For each one, write one sentence that begins:

```text
A teacher uses this when...
```

## Quality Check

Your definitions should help someone decide where a piece of content belongs. If every definition sounds like "a place for stuff," revise it.

## Safety / Accessibility / Source Note

Public websites and apps raise different safety questions than private course materials. Before publishing anything, check whether it includes student data, private teacher notes, copyrighted material, or links students cannot access.

## Capstone Connection

Your Starter Kit will include a project type decision. Name the container before you build it: course page, site, app, or reference resource. That one choice keeps the project from quietly turning into a polished pile of parts.
