## The Teacher Problem

Some teacher-builder work happens entirely in the browser. Some work needs software installed on the computer.

Confusing those two paths causes wasted time.

A teacher may try to install tools they do not need, or expect a browser editor to run commands it cannot run.

## The Idea

Browser tools run on a website. Local tools run on your computer.

| Workflow | Browser tool may be enough | Local tool usually needed |
|---|---|---|
| Write a lesson draft | Google Docs, LMS editor, GitHub web editor | VS Code if editing a repo locally |
| Review a source | browser | not usually |
| Edit many project files | possible but clunky | VS Code |
| Preview a web app locally | no | terminal plus project tools |
| Run tests or builds | usually no | terminal plus Node/npm or other tools |
| Use repo-aware agents | sometimes browser-based | often local or connected workspace |

## Classroom Scenario

A teacher can update a README in GitHub's browser editor. But to preview a site route before publishing, the teacher needs the project installed locally or a hosted preview environment.

Those are different jobs.

## Weak Version

```text
If a tool is in the browser, it is simpler and safer.
```

Not always. Browser tools can still publish, share, expose data, or edit the wrong file.

## Better Version

```text
I choose browser or local tools based on the job, the access needed, and the safety risk.
```

That is the right decision rule.

## Build Step

Add two columns to your workstation checklist:

```text
I can do this in the browser | I need local tools for this
```

Sort five tasks you expect to do, such as writing Markdown, reviewing GitHub files, previewing a course site, running a check, or editing images.

## Quality Check

If you cannot explain why a task belongs in one column, mark it `ask before doing`.

## Safety / Accessibility / Source Note

Browser-based does not mean risk-free. Check account ownership, sharing settings, public/private status, and whether student data is involved.

## Reflection

Which task have you been trying to do with the wrong kind of tool?
