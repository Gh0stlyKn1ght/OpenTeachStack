## The Teacher Problem

VS Code has many features, so beginners may miss the few that matter most.

For teacher-builder work, the first useful moves are search, preview, terminal, and careful extension choices.

## The Idea

Use each tool for its job.

| Feature | Teacher use | Safety check |
|---|---|---|
| Search | find titles, links, stale wording, or repeated terms | do not replace every match blindly |
| Preview | check Markdown layout before publishing | preview is not the same as full site QA |
| Terminal | run project commands | check the current folder first |
| Extensions | add helpful features | install only what you trust and need |

## Classroom Scenario

A teacher needs to find every mention of an old course title. Search finds the matches. The teacher edits only the current course files, then reviews changed files before committing.

## Weak Version

```text
Search found 30 matches, so replace all.
```

That can damage notes, reports, or unrelated courses.

## Better Version

```text
Search found candidates. I will inspect the file path and context before changing each one.
```

## Safety / Accessibility / Source Note

AI extensions and coding extensions can request broad access. Do not install them on a private or school-managed project without understanding the access boundary.

## Framework Purpose

This framework exists to keep **Search, Preview, Terminal, and Beginner Extensions** from turning into a pile of tips. A teacher should be able to use it when the work feels fuzzy: name the parts, make the next move smaller, and check whether the result is actually useful in a classroom.

## How to Use This Framework

Apply the framework in three passes:

1. Name the classroom job this tool or idea is supposed to do.
2. Match each part of the framework to one visible choice in your project.
3. Review the artifact for the obvious failure mode: polished wording with no evidence, unclear steps, missing source, or a tool choice that creates extra work for students or teachers.

Save the result in your Starter Kit as evidence of the decision, not as decoration.
