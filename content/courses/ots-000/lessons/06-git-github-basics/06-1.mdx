## The Teacher Problem

People say Git and GitHub together so often that beginners think they are the same thing.

They are related, but not identical.

## The Idea

Git tracks project history. GitHub hosts repositories online and supports collaboration.

| Term | Plain meaning | Teacher use |
|---|---|---|
| Git | version-control tool | track changes, compare versions, recover work |
| GitHub | website/platform for repos | store, share, review, collaborate, publish |
| Repository | project folder with history | hold course files and project notes |
| Commit | saved snapshot of changes | explain what changed and why |

## Classroom Scenario

A teacher edits curriculum locally. Git tracks the changes. GitHub stores the repo online so another teacher or agent can review the work.

## Weak Version

```text
GitHub saved my work, so I do not need to understand Git.
```

That hides the history layer.

## Better Version

```text
Git tracks the changes. GitHub gives the project an online home for review and collaboration.
```

## Build Step

Add these four terms to your GitHub checklist:

- Git
- GitHub
- repository
- commit

For each, write a teacher-friendly sentence.

## Quality Check

Your definitions should explain how the term protects or organizes project work.

## Safety / Accessibility / Source Note

Git can remember things you committed by mistake. Do not commit secrets, student data, or private files and assume deleting them later fully solves the problem.

## Reflection

Which idea matters more to you right now: history, online storage, collaboration, or recovery?
