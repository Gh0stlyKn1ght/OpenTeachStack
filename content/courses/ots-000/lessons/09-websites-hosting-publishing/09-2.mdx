## The Teacher Problem

Website words blur together.

Static site. Web app. Local preview. Published site. Build. Deploy.

When those words blur, teachers check the wrong thing.

## The Idea

Use a simple comparison.

| Term | Plain meaning | Teacher check |
|---|---|---|
| Static site | pages are generated and served mostly as files | do pages, links, and assets load? |
| Web app | site has more interactive behavior or server logic | do features work, not just pages? |
| Local preview | running on your computer | does it work before publishing? |
| Published site | available from a hosted URL | does it work for the intended audience? |

## Classroom Scenario

A teacher checks the local preview and sees the updated lesson. A colleague checks the published site and still sees the old version.

Both people are telling the truth. They are looking at different states.

## Weak Version

```text
I checked the site.
```

Which one?

## Better Version

```text
I checked the local preview. I still need to check the published URL after deployment.
```
