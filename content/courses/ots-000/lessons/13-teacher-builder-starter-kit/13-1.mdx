## The Teacher Problem

Capstones often pretend everyone has the same device, permissions, accounts, and time.

They do not.

## The Idea

Choose the setup you actually have, then write the boundaries honestly.

## Classroom Scenario

A teacher has a school Chromebook and a personal Windows laptop. The Chromebook is best for Docs, LMS, and GitHub reading. The Windows laptop is better for VS Code and local preview.

That is a real setup.

## Weak Version

```text
I need the perfect machine before I can begin.
```

No.

## Better Version

```text
I will use browser tools for planning and an approved local machine for project preview when needed.
```

## Build Step

Write your setup summary:

```text
Primary device:
Browser workflow:
Local workflow:
Accounts:
What is ready:
What is blocked:
What I will not do on this device:
```

## Quality Check

The setup should be believable for your actual work week.

## Safety / Accessibility / Source Note

Do not route around school policy just to complete a setup. Ask for approved access or choose a browser-based path.

## Reflection

What is one limitation you can plan around instead of fighting?
