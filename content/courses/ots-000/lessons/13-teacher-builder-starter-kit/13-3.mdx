## The Teacher Problem

The course should end with something built, not only vocabulary.

The starter Markdown lesson proves the teacher can create a small, structured content file.

## The Idea

Use the Markdown lesson from Chapter 08 and revise it for clarity.

## Classroom Scenario

A teacher writes a short lesson on evaluating a website source. The lesson includes target, directions, example, source note, and before-submit checklist.

It is small, but it is real.

## Weak Version

```text
I wrote a Markdown file with headings.
```

Headings alone do not teach.

## Better Version

```text
I wrote a short student-facing lesson that a student can follow and a teacher can revise.
```

## Build Step

Revise your starter lesson so it includes:

- title
- learning target
- directions
- example
- student check
- source or safety note
- revision note

## Quality Check

The lesson should answer: what should the student learn, do, and submit?

## Safety / Accessibility / Source Note

Use safe examples and descriptive links. Do not use real student work unless it is approved, anonymized, and necessary.

## Reflection

What did writing in Markdown make more visible about your lesson?

## Capstone Connection

This lesson is the sample content file inside the Teacher Builder Starter Kit.

## Artifact Build Scan

### Artifact

Build or revise **Write Your First Markdown Lesson** as a small working artifact, not a decorative note.

### Build Steps

1. Name the classroom job this artifact is supposed to do.
2. Fill in the artifact with the smallest useful version a teacher could reuse.
3. Add one piece of evidence that shows the artifact is usable.
4. Mark anything unfinished as revise or blocked instead of hiding it.

### Example Output

A finished example should be specific enough that another teacher can tell what changed, where it lives, and how it would be used in a real course build.

### Quality Check

- The artifact has a clear title.
- The teacher decision or classroom use is visible.
- The evidence is saved in a form someone else can inspect.
- The next revision trigger is named if the artifact is not ready.

### Capstone Connection

Save this artifact in the Teacher Builder Starter Kit. It becomes part of the evidence trail for deciding whether the starter kit is ready, needs revision, or is blocked.
