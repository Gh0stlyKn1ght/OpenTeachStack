## Maintenance rule

Software documentation changes. Before using a link in a lesson, template, or public release note:

1. Open the official source.
2. Check whether the page is current enough for the claim.
3. Prefer install/setup pages from official vendors.
4. Avoid relying on AI summaries for commands, permissions, pricing, or security behavior.
5. Record the date checked when a lesson depends on the source.
