# OTS-101 Drafts

This folder is the safe workbench for proposed OTS-101 changes.

Drafts are not course source. They are reviewable proposals that may later be promoted into `content/courses/ots-101` through an explicit promotion workflow.

Use this folder for:

- proposed lesson revisions
- proposed lab or template updates
- authoring notes
- validation reports
- review evidence

Do not use this folder to mark OTS-101 live, reviewed, or release-ready.
