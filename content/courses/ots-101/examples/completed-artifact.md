# Completed Artifact

## Completed Teacher Workflow Audit Sample

This sample shows a first-pass workflow audit for a beginner-friendly mini-unit. It uses fictional classroom details and sample-only student evidence.

## Workflow Summary

| Field | Completed example |
| --- | --- |
| Workflow name | Sensor-threshold mini-unit setup |
| Course context | Grade 8 robotics |
| Source or standard | Students collect sensor data, identify patterns, and justify a design decision |
| Student-facing task | Teams collect five distance readings and choose a stop threshold for a robot |
| Evidence students produce | Data table with units, threshold decision, and one reflection sentence |
| Files or tools used | Slide directions, sensor setup photo, response sheet, sample data row, teacher revision note |
| Verification before teaching | Test one sensor, confirm centimeter units, check the response sheet has a sample row |
| Revision note after teaching | Add a second sample row if teams confuse centimeters and inches |

## Weak And Strong Entries

| Weak entry | Stronger entry |
| --- | --- |
| Use the sensor worksheet. | Students use the current response sheet to record five centimeter readings and mark one unreliable reading. |
| Check the lesson before class. | Teacher tests one sensor, opens the slide deck, confirms the sample row has units, and removes the old inches table. |
| Save student work. | Archive one fictional sample response and a revision note; keep real student work private. |

## Privacy And Safety Notes

- Use fictional sample student names when sharing the workflow publicly.
- Do not publish real rosters, student photos, grades, private comments, or school-only file links.
- Keep answer keys and sensitive setup notes in the private teacher folder.
- Label public sample rows as sample-only so another teacher does not confuse them with real student evidence.

## Reviewer Check

Before calling the workflow audit ready, another educator should be able to answer yes to each question:

- Can I tell what students do during class?
- Can I find the source, student-facing artifact, and evidence?
- Can I see the check the teacher runs before teaching?
- Can I tell what changes after the first classroom use?
- Can I identify what stays private?

Use this as a minimum baseline before class launch. A stronger artifact includes a second workflow row from a different subject area and one dated revision note from an actual pilot.

## Standards Unpacking Row

| Field | Completed example |
| --- | --- |
| Standard or source language | Students evaluate reliability of measurements. |
| Learning target | I can compare repeated sensor readings, identify one unreliable value, and justify a stop-threshold decision. |
| Student product | Data table with units, threshold decision, and four-sentence explanation. |
| Evidence of readiness | Student cites at least two readings, explains why one reading may be unreliable, and revises after feedback. |
| Revision trigger | Revise the target if students can complete the table but cannot explain the unreliable reading. |

## Mini-Unit Map

| Lesson | Student evidence | Review rule |
| --- | --- | --- |
| 1. Record readings | Five-row data table with units | Keep if all rows include centimeters. |
| 2. Identify reliability issue | One flagged reading and reason | Revise if students flag without evidence. |
| 3. Choose threshold | Claim and evidence chart | Keep if claim cites two readings. |
| 4. Revise from feedback | Revised threshold and peer note | Pause if peer feedback is only praise. |
| 5. Reflect and archive | Reflection plus teacher update log | Keep if revision is visible. |

## Resource Evaluation Rows

| Resource | Decision | Reason |
| --- | --- | --- |
| Official sensor setup guide | Keep | Use for teacher setup; cite title, publisher, and access date. |
| OER robotics data-table activity | Revise | Keep structure but replace examples with fictional classroom-safe rows. |
| Personal blog tip | Reject for student-facing use | No source trail, unclear license, and examples do not match materials. |

## Observable Rubric Rows

| Criterion | Ready evidence |
| --- | --- |
