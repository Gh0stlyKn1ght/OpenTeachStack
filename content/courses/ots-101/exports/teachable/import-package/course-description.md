# Teaching Teachers Foundations

## Audience
Beginning teachers and instructional coaches

## Promise
Build a reusable, reviewable mini-unit workflow through prompting, AI verification, standards alignment, resource evaluation, assessment, and delivery planning.

## Prerequisites
No prior coding required. Bring one target unit and one school-safe student artifact.

## Expected Learner Outcomes
By the end of the course, learners can:
- write safer lesson-planning prompts with review triggers,
- verify AI output against sources, fake-data tests, and privacy boundaries,
- translate one standard into a measurable learning target and evidence row,
- evaluate resources with source, license, accessibility, and classroom-fit notes,
- build a 3-5 lesson mini-unit map,
- write observable rubric criteria,
- prepare 45-minute and block-period delivery plans with fallbacks,
- assemble a final capstone packet another educator can inspect.

## Artifacts Produced
- Teacher workflow audit
- Reusable prompt library
- AI verification checklist
- Standards unpacking sheet
- Resource evaluation sheet
- Mini-unit map and lesson template
- Assessment and rubric draft
- Delivery plan
- Verified mini-unit capstone packet

## Final Project
A verified mini-unit capstone packet with artifact links, ready/revise/blocked decisions, privacy notes, review owner, and maintenance trigger.

## Safety / Privacy / Accessibility
Use fictional students in examples. Do not upload real student data, private school links, student photos, grades, rosters, or identifiable classroom notes. Public-facing samples should use fictional rows and clear source/license notes.

## Assessment and Review
Short checks after each chapter artifact row, ready/revise/blocked review language, peer-review prompts, and a final capstone release decision.
