# Instructor Video Script Notes

## Lesson Start (45-60 sec)
- State the chapter outcome in one sentence.
- Name what artifact is the focus.

## Core Segment
- Demonstrate one concrete example for the chapter.
- Pause for a check question at each decision point.

## End Segment
- Show artifact quality check.
- Share the next revision step and reason.

## Accessibility
- Keep slides visually simple.
- Call out any file names on screen and include pauses for note-taking.
