# Tools vs Workflows vs Systems

## Why This Matters

A teacher building a first reusable mini-unit needs a system another educator can inspect. The easiest mistake is to start with tools: a learning management system, a folder, an AI chatbot, a spreadsheet, a slide deck. Tools matter, but they do not tell you what teaching decision has been made.

This lesson separates three ideas that often get blended together: tools, workflows, and systems.

## Three Levels

A **tool** is something you use. Examples: Google Docs, Canvas, a sensor kit, a rubric generator, a spreadsheet, a prompt library.

A **workflow** is a repeated sequence of actions. Examples: collect lab data, review drafts, publish weekly directions, score exit tickets, revise a lesson after teaching.

A **system** connects workflows so the teaching work can be used, checked, and improved. It names the source, the task, the evidence, the storage location, the review rule, and the revision habit.

## Why The Distinction Matters

Tool-first thinking often sounds productive:

> We will use Google Forms for exit tickets.

Workflow thinking is clearer:

> Students answer three exit-ticket questions. The teacher reviews the misconception question before the next class and tags two reteach groups.

System thinking is better still:

> The exit-ticket workflow lives in the Unit 2 folder. The form links to the day's learning target, responses flow to the tracker, the teacher reviews misconceptions before the next lesson, and the revision note records whether the next lesson needs a warm-up repair.

The tool did not disappear. It became part of a teachable system.

## Classroom Example

For the robotics mini-unit:

| Level | Example |
| --- | --- |
| Tool | Ultrasonic sensor, slide deck, shared response sheet |
| Workflow | Students measure distances, record data, choose a stop threshold, and explain the choice |
| System | The source standard, safety setup, student directions, response sheet, sample data, teacher check, and revision note are stored together |

If the teacher only names the tool, another educator still has to guess the lesson. If the teacher names the system, another educator can inspect the plan and ask useful questions.

## Practice

Pick one tool you use often. Then complete this sentence set:

- Tool: We use ...
- Workflow: Students or teachers use it to ...
- System connection: This connects to the source, evidence, storage, review, and revision routine by ...

Example:

- Tool: We use a shared Google Doc.
- Workflow: Teams record sensor readings and write a threshold claim.
- System connection: The doc is linked from the unit directions, copied from a locked template, checked for units before class, and archived with a revision note after the lesson.
