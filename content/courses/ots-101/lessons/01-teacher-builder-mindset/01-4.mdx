# Source -> Prompt -> Build -> Verify -> Teach -> Archive -> Improve

This course uses a repeatable teacher-builder loop:

Source -> Prompt -> Build -> Verify -> Teach -> Archive -> Improve

The loop is meant to slow down the risky parts of AI-assisted curriculum work without making the teacher move like a committee. Each step leaves enough evidence that the work can be trusted, revised, and reused later.

## The Loop

Source means starting from something accountable: a standard, official documentation, a district requirement, a primary source, a classroom constraint, or a credible reference.

Prompt means giving the AI or planning tool a job with context, constraints, audience, format, and revision rules. A prompt is not a wish. It is a work order.

Build means turning the draft into a usable artifact: a lesson page, rubric, slide sequence, checklist, example set, lab, or teacher note.

Verify means checking facts, links, privacy, accessibility, copyright, and fit for the actual students who will use the material.

Teach means using the artifact in a real instructional setting, not treating the draft as finished just because it looks polished.

Archive means saving the final version, source notes, and related assets where the course can find them later.

Improve means recording what happened and making one revision while the teaching experience is still fresh.

## Do This

Take one small planning task and run it through the loop. Do not choose a whole unit yet. Choose something compact, such as one set of student directions, one warm-up, one rubric row, or one source note.

Write one sentence for each step:

- Source: What did you start from?
- Prompt: What did you ask for?
- Build: What artifact did you create?
- Verify: What did you check?
- Teach: Where would students encounter it?
- Archive: Where would the final version live?
- Improve: What would you review after using it?

## Evidence of Completion

You should have a seven-line workflow note. The note does not need to be beautiful. It needs to make the teacher decision trail visible.

## Verification Check

If an AI tool was involved, do not paste student data, private staff information, hidden assessment keys, or account credentials into the prompt. Replace sensitive context with safe placeholders.
