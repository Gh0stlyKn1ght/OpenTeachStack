# Chapter Plan

## Why This Matters

A teacher building a first reusable mini-unit needs a system another educator can inspect. Prompting belongs in that system because an AI response is not a lesson plan by itself. A prompt is a work order: it tells the tool what role to play, what context matters, what source material to use, what output to create, and how the teacher will review the result.

This chapter builds a **Reusable prompt library** for teacher planning work. The goal is not to collect clever prompts. The goal is to create prompts that protect teacher judgment, student privacy, source accuracy, and classroom usability.

## Chapter Situation

Use the same mini-unit or workflow from Chapter 1. The model here is a grade 8 robotics activity where students use ultrasonic sensor readings to choose a robot stop threshold.

You might use AI to draft:

- student-facing directions,
- a data-table template,
- a misconception check,
- rubric criteria,
- a parent-friendly explanation,
- revision suggestions after reviewing student work.

Each of those prompts needs boundaries. The AI should not invent standards, cite fake sources, receive private student details, or replace the teacher's decision.

## Chapter Map

You will learn how to:

1. Explain what a prompt is and what it cannot do.
2. Build prompts from role, context, task, source, constraints, and review criteria.
3. Compare weak prompts with useful prompts.
4. Draft prompts for lesson plans, rubrics, quizzes, and student directions.
5. Revise AI output so it sounds like a teacher and fits the classroom.
6. Store reusable prompts with notes about when to use them.

## First Prompt Library Row

Start your prompt library with a planning prompt.

| Field | Entry |
| --- | --- |
| Prompt name | Draft a student direction sheet |
| Use case | Turn a teacher task description into student-facing directions |
| Teacher context needed | Grade, time, materials, learning target, student product, safety notes |
| Privacy rule | Use no student names or private class details |
| Verification rule | Teacher checks source alignment, reading level, and classroom feasibility |

## Quality Check

By the end of this chapter, your prompt library should include at least three reusable prompts:

- one for planning or directions,
- one for assessment or feedback,
- one for revision or verification.

Each prompt should include a teacher review step. If the review step is missing, the prompt is not ready for classroom use.
