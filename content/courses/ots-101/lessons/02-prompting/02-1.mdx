# What Prompting Actually Is

## Why This Matters

A teacher building a first reusable mini-unit needs a system another educator can inspect. Prompting is part of that system when it records the request that shaped a draft, the limits placed on the tool, and the review the teacher performed afterward.

Prompting is not magic phrasing. It is structured communication. A useful prompt tells an AI tool what job to do and what job not to do.

## A Prompt Is A Work Order

When you give a contractor a work order, you do not say, "Fix the room." You name the room, the problem, the materials, the deadline, and the inspection rule.

Prompting works the same way. A teacher prompt should usually name:

- role: what perspective the tool should use,
- context: grade, course, unit, time, materials,
- task: what output you need,
- source: what information should anchor the response,
- constraints: reading level, format, length, safety limits,
- review: what the teacher will check before use.

## Weak And Strong Versions

Weak:

> Make a robotics lesson.

Stronger:

> Act as a middle school technology curriculum assistant. Draft student-facing directions for a 45-minute ultrasonic sensor activity. Students will collect five distance readings in centimeters, identify one reading that may be unreliable, choose a robot stop threshold, and write a two-sentence justification. Use plain grade 8 language. Do not invent standards or cite sources. Include a teacher verification checklist at the end.

The stronger prompt gives the tool a job, but it also protects the teacher's job. The teacher still checks accuracy, tone, safety, alignment, and feasibility.

## Practice

Take one prompt you have used or might use. Rewrite it as a work order.

Use this frame:

- Role: Act as ...
- Context: My class is ...
- Task: Create ...
- Source: Use only ...
- Constraints: Keep it ...
- Review: Include ...

## Add To The Prompt Library

Add your rewritten prompt to the **Reusable prompt library**. Include a note explaining when to use it and when not to use it.

Example limit:

> Do not use this prompt with real student names, behavior details, IEP information, grades, or private family context.

## Quality Check

Your prompt is useful if another teacher can see what the AI was asked to do, what information it was allowed to use, and what the teacher must verify.
