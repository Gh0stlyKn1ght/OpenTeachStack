# Prompt Anatomy

## Why This Matters

A teacher building a first reusable mini-unit needs a system another educator can inspect. Prompt anatomy gives you a way to inspect AI-assisted work instead of guessing why a response was useful or useless.

When a prompt has missing parts, the AI fills gaps with assumptions. Sometimes those assumptions are harmless. Sometimes they create fake alignment, vague student tasks, inaccessible directions, or confident errors.

## The Six Parts

Use this anatomy for most teacher prompts:

| Part | Purpose | Example |
| --- | --- | --- |
| Role | Sets the lens | Act as a middle school technology curriculum assistant |
| Context | Names the class situation | Grade 8 robotics, 45 minutes, sensor kits |
| Task | Names the output | Draft student directions |
| Source | Anchors claims | Use the learning target and teacher notes below |
| Constraints | Controls shape and safety | Use plain language, no private student data |
| Review criteria | Tells what to check | Include a checklist for units, timing, and evidence |

Not every prompt needs a long paragraph, but every classroom-ready prompt needs enough anatomy to protect the work.

## Annotated Example

> **Role:** Act as a curriculum assistant for a middle school robotics teacher.
> **Context:** Students are learning how ultrasonic sensors report distance in centimeters. They have 45 minutes and work in teams of three.
> **Task:** Draft student-facing directions for collecting five readings and choosing a robot stop threshold.
> **Source:** Use this learning target: "I can use sensor data to justify a design decision."
> **Constraints:** Use grade 8 language, include a small data table, and do not invent standards or citations.
> **Review criteria:** End with a teacher checklist for setup safety, units, timing, and evidence of learning.

This prompt gives enough structure that the teacher can judge the response against the original request.

## Practice

Choose a weak prompt and diagnose the missing parts.

Weak prompt:

> Make a quiz about sensors.

Diagnosis:

- Role missing
- Grade and time missing
- Source missing
- Question types missing
- Student evidence unclear
- Review criteria missing

Rewritten prompt:

> Act as a grade 8 robotics assessment assistant. Create a five-question check for understanding about ultrasonic sensor readings. Use the learning target "I can use sensor data to justify a design decision." Include three multiple-choice questions, one short data-table interpretation, and one short written justification. Mark the answer key as teacher-only and include a note about what misconception each question checks.

## Quality Check

Your prompt is ready to test when a colleague can identify all six parts without asking you what you meant.
