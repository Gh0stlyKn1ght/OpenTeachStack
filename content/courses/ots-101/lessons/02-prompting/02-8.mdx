# Prompt Library

## Why This Matters

A teacher building a first reusable mini-unit needs more than a tool list or a checklist. The goal in **Prompting for Teachers** is to make a classroom system that can be used, inspected, revised, and handed to another educator without needing a meeting first.

For **Prompt Library**, the practical pressure is simple: accepting a polished lesson draft before the sources, student task, assessment evidence, and classroom routine are inspectable. This page turns that pressure into a visible update to **Reusable prompt library**.

## Classroom Scenario

Use this situation as the working example: an ultrasonic sensor mini-unit where students collect readings, explain uncertainty, revise a threshold, and submit a short design reflection. By the end, the work should support this habit: source-backed planning, plain-language decisions, and visible revision notes.

In this scenario, the learner should be able to point to:

- what the teacher prepares before students see the work,
- what students will read, build, submit, or discuss,
- what evidence shows the work is ready,
- what gets revised after the first use.

## Key Idea

A prompt library is useful only when each saved prompt includes a purpose, reuse note, source requirement, revision date, and a short example of the expected output.

Watch for this mistake: Saving a pile of prompts without naming when to use them, what source material they require, or how to judge the output.

- Before class, connect prompt library to one setup decision in the reusable prompt library: what the teacher prepares, where it lives, and which source or safety check keeps it honest.

## What Good Work Looks Like

A strong **prompt library** entry is concrete enough that another teacher can inspect it. It names the class context, the artifact location, the student product, the source or safety check, and the revision trigger.

For this lesson, a strong entry might say:

> Library entry: Sensor data-table prompt / use during lab planning / requires current mini-unit notes and datasheet / revise after first student confusion note / sample output attached.

That level of detail is enough to review. It is also small enough to revise after one classroom use.
