> OTS-101 frame: AI supports teacher-approved course content that becomes student-facing only after review.

Good student directions reduce invisible work.

They do not make the learning easy. They make the task understandable so students can spend their energy on the actual thinking.

A strong direction usually includes four parts:

**Use. Do. Make. Check.**

<FrameworkBlock title="Use, do, make, check" eyebrow="Direction pattern">
  <ConceptCard title="Use">
    Name the source, example, tool, file, notes, or material students need.
  </ConceptCard>
  <ConceptCard title="Do">
    Name the action students should take with that material.
  </ConceptCard>
  <ConceptCard title="Make">
    Name the product, response, submission, or evidence students will create.
  </ConceptCard>
  <ConceptCard title="Check">
    Name how students can tell whether their work is complete or strong.
  </ConceptCard>
</FrameworkBlock>

## Use

Start by naming the material.

Weak:

> Look at the examples.

Better:

> Open the two sample responses labeled Response A and Response B.

The better version removes guessing.

## Do

Tell students the action.

Weak:

> Think about which one is better.

Better:

> Highlight one sentence in each response that uses evidence.

The better version turns thinking into visible work.

## Make

Name the product.

Weak:

> Be ready to discuss.

Better:

> Write a three-sentence explanation choosing the stronger response.

The better version tells students what evidence of learning will exist.

## Check

Give students a quality marker.

Weak:

> Do your best.

Better:

> Before submitting, check that your explanation names the stronger response, includes one piece of evidence, and explains why that evidence matters.

The better version gives students a way to revise before the teacher grades it.

Add one support move when the task has a predictable stuck point. This does not need to become a long help section. A single line can prevent a lot of quiet confusion.

Example:

```text
If you are stuck, reread the first sentence of each source and look for who created it.
```

## Put the pattern together

Here is a complete direction:

```text
Use Response A and Response B. Highlight one sentence in each response that uses evidence. Then write a three-sentence explanation choosing the stronger response. Before submitting, check that your explanation names the stronger response, includes one piece of evidence, and explains why that evidence matters.
```

That direction is not flashy.

It is usable.

## Ask AI for directions using the pattern

You can use your Chapter 3 prompt to draft student directions.

```text
Using the learning target and teacher notes below, draft student-facing directions using the Use-Do-Make-Check pattern. Keep the language direct and age-appropriate. Do not add tools or materials that are not listed. End with one teacher-check note naming anything I should verify before publishing.
```

Then inspect the output with your Chapter 4 checklist.

## Classroom Scenario

A student opens a lesson after being absent. The page says:

> Compare the examples and submit your answer.

The teacher knows which examples, what comparison, and where to submit. The student does not.

## Weak Version

Weak direction:

> Review the sources and answer the questions.

It names a general school activity, but not the material, action, product, or quality check.

## Better Version

Better direction:

> Use Source A and Source B. Underline one credibility clue in each source. Then write four sentences choosing the source you trust more and explaining two clues that support your choice. Before submitting, check that your answer names both clues.

Now the direction carries the work.

## Build Step

Rewrite one direction using this frame:

```text
Use:
Do:
Make:
Check:
Stuck point:
```

## Quality Check

The direction is ready when a student can start without asking which file, which action, which product, or which quality marker you meant.

It is stronger when a student also knows one reasonable next move before waiting for the teacher.

## Safety / Accessibility / Source Note

If directions depend on a link, file, video, or tool, check access before publishing. Clear wording cannot rescue a resource students cannot open.

## When This Helps

Use this pattern when a lesson page has directions that make sense to you but still require live teacher translation for students.

## Steps

1. Name the material students use.
2. Name the action students take with it.
3. Name the product or evidence students make.
4. Name the check students use before submitting.
5. Remove any step that depends on hidden teacher explanation.
