# Design Assignments and Labs That Prove the Target

> OTS-101 frame: AI supports teacher-approved course content that becomes student-facing only after review.

A strong assignment starts with the learning target, not the product format.

Do not start with "make a worksheet" or "make a lab."

Start with:

> What evidence would prove students can do the thing we taught?

Then choose the task format that produces that evidence.

## Step 1: Copy the target

Put the learning target at the top of the assignment draft.

Example:

```text
Learning target: I can choose the more trustworthy source and explain my choice using evidence.
```

This keeps the assignment from drifting into a fun but unrelated activity.

## Step 2: Name the evidence

Ask what the teacher needs to see.

For the source credibility target, evidence might be:

- source choice
- two credibility clues
- short explanation
- citation or link to each source

That is more useful than simply saying "students discuss credibility."

## Step 3: Choose the task type

Choose the simplest task that produces the evidence.

For example:

- comparison chart
- short written explanation
- annotated screenshot
- lab observation table
- troubleshooting log
- reflection with evidence
- revised draft with notes

The task type should serve the target.

## Step 4: Write student-facing directions

Use the pattern from Chapter 5.

```text
Use the two source links provided. Compare the author, date, evidence, and purpose of each source. Choose the source you would trust more for a class project. Write four sentences explaining your choice with two credibility clues.
```

This gives students materials, action, product, and success conditions.

## Step 5: Verify fit

Before publishing, check:

- Does the task require the target skill?
- Is the evidence visible?
- Are students using approved materials?
- Is the task realistic for the time available?
- Can students access the tools?
- Does the task avoid private student data?

## Ask AI for a bounded task

Use AI to draft, not decide.

```text
Using this learning target and source list, draft one student-facing assignment that produces visible evidence of the target. Include directions, required submission, and a teacher-check note. Do not add sources, tools, or standards that are not listed.
```

Then review with your verification checklist.

<TakeawayStrip tone="teacher">
  The assignment is not the evidence because it exists. It becomes evidence when the task requires students to show the target skill.
</TakeawayStrip>
