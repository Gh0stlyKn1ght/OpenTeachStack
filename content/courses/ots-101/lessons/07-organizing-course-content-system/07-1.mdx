> OTS-101 frame: AI supports teacher-approved course content that becomes student-facing only after review.

A platform page is not automatically the course source.

A slide deck is not automatically the course source.

A handout, PDF, LMS post, or copied Google Doc is not automatically the course source either.

Those may be exports: versions of the course content adapted for delivery.

## Why This Matters in Practice

The idea becomes useful only when it changes what a teacher saves, revises, or gives to students.

The source of truth is the teacher-approved place where the current lesson content is maintained.

## Why the distinction matters

When the source and the export get mixed up, updates become dangerous.

A teacher fixes the lesson in one place, but the LMS still has the old version. Or the LMS is updated, but the planning file still says something else. Or an AI draft gets pasted into the platform and no one knows whether it was reviewed.

That is how two versions of the course start fighting each other.

## What can be a source of truth?

A source of truth can be simple.

It might be:

- a folder of Markdown files
- a Google Drive folder with approved lesson documents
- a curriculum planning document
- a versioned repository
- a course authoring system
- a shared team folder with review rules

The tool matters less than the rule:

> This is the place we trust first.

## What counts as an export?

Exports include anything adapted from the source for a specific use.

Examples:

- LMS page
- student handout
- slide deck
- PDF
- website page
- printed packet
- copied email announcement
- quiz import file
- calendar post

Exports are important, but they should not become mystery sources.

## The teacher-friendly rule

Use this rule:

```text
If I need to fix the lesson, I update the source first.
If students need to use the lesson, I publish or update the export.
```

That rule keeps the course from splitting into local version, network version, platform version, and random AI version.

## Add a source note to exports

When possible, label exports with their source.

Example:

```text
Source: Unit 2 / Lesson 3 / evidence-comparison.md
Export: Canvas page updated 2026-06-26
```

That small note can save hours later.

<TakeawayStrip tone="warning">
  If no one knows which version is the source, every version becomes suspicious.
</TakeawayStrip>

## Problem

Source of Truth Is Not the Same as Platform Export matters because AI can make weak course work look complete. The teacher still has to decide what students should learn, what evidence counts, and what belongs in front of the class.

## Common Misunderstanding

The common mistake is thinking a polished AI draft means the curriculum decision is finished. A draft can be useful and still be unready for students.

## Concrete Example

Instead of asking AI to "make a lesson," the teacher names the learning target, the student task, and the evidence to check. AI can then help draft language inside those boundaries.
