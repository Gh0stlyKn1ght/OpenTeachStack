# Use Sources and Media Responsibly

> OTS-101 frame: AI supports teacher-approved course content that becomes student-facing only after review.

Course content often depends on sources: articles, videos, images, datasets, screenshots, examples, documents, and websites.

AI can help summarize and adapt source material, but the teacher still needs to check whether the source is appropriate, trustworthy, and used responsibly.

## Source quality comes first

Before using a source with students, ask:

- Who created it?
- Why was it created?
- Is it current enough for the topic?
- Does it support the claim or activity?
- Is it appropriate for the grade and context?
- Can students access it safely?
- Does it require an account, payment, or personal data?

If a source fails those checks, do not build the lesson around it.

## AI summaries need source checks

If AI summarizes an article, video, policy, or reading, compare the summary against the source.

Ask:

- Did the summary keep the meaning?
- Did it leave out an important limitation?
- Did it add claims that are not in the source?
- Did it oversimplify in a misleading way?
- Does the summary need a citation or link?

A smooth summary is not automatically accurate.

## Copyright and reuse basics

This course is not legal advice.

The teacher-friendly rule is simpler:

> Know what you are using, why you are using it, and whether your use is appropriate for your classroom context.

Be careful with:

- copying full articles
- using images from random web searches
- reposting paid materials
- removing attribution
- uploading copyrighted media into public spaces
- asking AI to recreate protected work
- using content without checking license or permission context

When possible, prefer materials that are openly licensed, teacher-created, public domain, school-approved, or linked rather than copied.

## Attribution helps

Attribution does not solve every copyright issue, but it is a good habit.

Track:

- title
- creator or organization
- link
- license or usage note when available
- date accessed if useful
- how the source is used in the lesson

A small source note makes later review easier.

## Do not let AI invent sources

If AI gives a citation, link, author, or title, verify it before using it.

Better yet, provide the sources yourself and ask AI to work only from those sources.

```text
Use only the sources listed below. Do not add citations, facts, or links. If the source does not support a claim, say that it is not supported.
```
