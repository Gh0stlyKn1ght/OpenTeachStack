# Content Safety Review

## Purpose

Use this to check content before release.

## When to use it

Use it before publishing to students, an LMS, a website, Teachable, or a public course hub.

## Quality check

- Each fix has an owner.
- Anything unresolved is not published.
- Anything blocked is named honestly.
- Source claims are verified or removed.

## Safety note

Public sharing increases risk. Review links, permissions, names, images, and embedded files before release. If release evidence is missing, mark the item blocked instead of ready.
