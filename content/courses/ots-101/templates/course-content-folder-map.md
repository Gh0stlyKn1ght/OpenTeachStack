# Course Content Folder Map

## Purpose

Use this to make course files findable and maintainable.

## When to use it

Use it before publishing or sharing course content with another teacher.

## Blank fields

| Folder | Purpose | Audience | Naming rule | Archive rule |
| --- | --- | --- | --- | --- |
|  |  |  |  |  |

## Completed fictional example

| Folder | Purpose | Audience | Naming rule | Archive rule |
| --- | --- | --- | --- | --- |
| 01-student-facing | Lessons, assignments, checklists students use | Students | `unit-lesson-title-v1` | Move old versions to archive after revision |
| 02-teacher-materials | Demo notes, setup list, answer keys | Teacher | `teacher-only-topic-date` | Never publish publicly |

## Quality check

- Student-facing and teacher-only materials are separated.
- Archive rules are written down.
- File names help another teacher understand the content.

## Safety note

Keep answer keys, private notes, and student data out of public or student-shared folders.
