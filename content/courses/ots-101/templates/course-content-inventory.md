# Course Content Inventory

## Purpose

Use this before asking AI to revise course materials. It separates curriculum decisions, student-facing course content, instructional materials, platform locations, and verification needs.

## When to use it

Use it when a unit has scattered files, old lesson drafts, mixed teacher notes, or AI output that has not been sorted.

Use it especially when AI has generated a course package that looks complete because it includes descriptions, outlines, quiz plans, download lists, or platform notes.

## Quality check

- Every item has a category.
- Student-facing items are separate from teacher-only materials.
- OTS-000 context is included only when it clarifies tools, access, source, platform, or safety.
- AI tasks are specific enough to prompt safely.
- Verification tasks stay with the teacher.
- Export or platform items do not count as proof that the source course is ready.

## Safety note

Do not paste real student data, private links, grades, accommodations, or internal policy text into AI tools while filling this inventory.
