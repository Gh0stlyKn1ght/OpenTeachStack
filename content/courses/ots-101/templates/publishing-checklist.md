# Publishing Checklist

## Purpose

Use this to move course content into a platform without losing the course structure.

This checklist is for teacher-level publishing review. It is not a site build, DNS, deployment, or automation plan.

## When to use it

Use it before publishing to Google Classroom, Canvas, Teachable, a website, or a course hub.

## Quality check

- Students can find the next action.
- Teacher-only materials are not public.
- Links work from a student account.
- Platform copies point back to the approved source.
- Missing student-view checks are marked blocked, not ready.

## Safety note

Check file permissions from a student view before release. If student view is unavailable, mark the publishing decision blocked.
