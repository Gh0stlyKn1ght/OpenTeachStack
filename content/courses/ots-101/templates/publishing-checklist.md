# Publishing Checklist

## Purpose

Use this to move course content into a platform without losing the course structure.

## When to use it

Use it before publishing to Google Classroom, Canvas, Teachable, a website, or a course hub.

## Blank fields

| Item | Public | Classroom-only | Teacher-only | Link checked | Notes |
| --- | --- | --- | --- | --- | --- |
|  | yes / no | yes / no | yes / no | yes / no |  |

## Completed fictional example

| Item | Public | Classroom-only | Teacher-only | Link checked | Notes |
| --- | --- | --- | --- | --- | --- |
| Startup lesson page | no | yes | no | yes | Posted in Google Classroom. |
| Teacher setup notes | no | no | yes | yes | Stored in private Drive folder. |
| Safety checklist | no | yes | no | yes | Text version included. |

## Quality check

- Students can find the next action.
- Teacher-only materials are not public.
- Links work from a student account.

## Safety note

Check file permissions from a student view before release.
