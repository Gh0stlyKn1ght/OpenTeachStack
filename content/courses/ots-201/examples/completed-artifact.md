# Completed Artifact

## Completed Workspace Command Center Sample

This sample shows a Workspace system another teacher can inspect during a busy week.

## Drive Architecture Map

| Folder role | Model folder | Access rule |
| --- | --- | --- |
| Course root | `Robotics - 2026` | Teacher edit, teammate comment |
| Active unit | `Unit 03 - Sensors - ACTIVE` | Teacher edit |
| Student-facing materials | `Student Links - Unit 03` | Student view |
| Teacher planning | `Teacher Planning - Unit 03` | Teacher only |
| Evidence | `Evidence - Unit 03` | Teacher only unless sample is fictional |
| Archive | `Archive - 2025 and Earlier` | Teacher edit, not student-facing |
