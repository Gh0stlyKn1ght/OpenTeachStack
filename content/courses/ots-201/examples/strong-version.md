# Strong Version

## Strong Workspace Artifact

| Artifact | Strong evidence |
| --- | --- |
| Drive architecture map | Folder roles, names, access rules, archive rule, and review trigger are visible. |
| Reusable delivery template | Directions Doc and Slide deck roles are separated, with evidence path and revision log. |
| Curriculum tracker | Rows show status, student link, evidence link, owner, and next action. |
| Forms/Calendar workflow | Form purpose, response location, calendar trigger, and closeout rule are named. |
| Command center | Current unit, student links, evidence, operations, maintenance, revision log, and access notes are visible. |

## Strong Review Pattern

Use these decisions:

- **Ready:** another educator can use the workspace without asking which file is current.
- **Revise:** the workspace is usable but one owner, status, evidence link, or permission note is missing.
- **Blocked:** student-facing links expose private evidence, answer keys, or old materials.
