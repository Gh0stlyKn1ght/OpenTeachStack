# Folders by Course, Unit, and Year

## Why This Matters

Most Drive confusion is time confusion. A file may be useful, but is it the current version, last year's version, a draft for next term, or a shared team template?

A workspace system works when folder names reveal course, unit, and year without relying on memory.

## Core Idea

Use names that show scope and time.

Good Drive names do not need to be clever. They need to sort, scan, and prevent old material from looking current.

## Model Naming Pattern

| Level | Pattern | Example |
| --- | --- | --- |
| Course root | `Course - Year` | `Robotics - 2026` |
| Unit folder | `Unit ## - Topic - STATUS` | `Unit 03 - Sensors - ACTIVE` |
| Student-facing folder | `Student Links - Unit ##` | `Student Links - Unit 03` |
| Teacher planning | `Teacher Planning - Unit ##` | `Teacher Planning - Unit 03` |
| Evidence folder | `Evidence - Unit ##` | `Evidence - Unit 03` |
| Archive | `Archive - Year or Term` | `Archive - 2025 and Earlier` |

Status labels should be plain:

| Label | Meaning |
| --- | --- |
| `ACTIVE` | Use this now. |
| `READY` | Prepared, not yet taught. |
| `REVISE` | Keep, but update before reuse. |
| `CLOSED` | Finished for students. |
| `ARCHIVE` | Stored for reference only. |

## Classroom Example

Maya replaces this unclear folder set:

| Old folder | Problem |
| --- | --- |
| `Robotics Stuff` | No course year, status, or role. |
| `New Sensors` | Could be draft or active. |
| `Copy of Unit 3` | No one knows why the copy exists. |

With this structure:

```text
Robotics - 2026
  00 Course Admin
  01 Team Shared Templates
  Unit 03 - Sensors - ACTIVE
    Student Links - Unit 03
    Teacher Planning - Unit 03
    Evidence - Unit 03
  Archive - 2025 and Earlier
```

## Verification Checklist

- Folder names show course, unit, and year where needed.
- Current work uses a visible status label.
- Draft, closed, and archived material cannot be mistaken for active teaching.
- Student-facing files have their own clear location.
- Evidence is not mixed with teacher drafts.
- The naming pattern is simple enough to repeat next unit.

## Reflection

Which existing folder name hides time or status? Rewrite the name before moving any files.
