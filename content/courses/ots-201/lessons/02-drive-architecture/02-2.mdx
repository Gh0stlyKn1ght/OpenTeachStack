# Shared Drives and Permissions

## Why This Matters

Folder structure and sharing rules are the same decision. A beautiful folder map can still fail if students can edit teacher materials or a teammate cannot access the shared template they are supposed to maintain.

A workspace system works when permissions match the job each person actually needs to do.

## Core Idea

Choose access by responsibility.

Do not ask "Who might need this someday?" Ask "What must this person do with this folder or file?"

## Permission Roles

| Person or group | Typical access | Why |
| --- | --- | --- |
| Teacher owner | Edit | Maintains current materials and archives. |
| Co-teacher or teammate | Comment or edit, depending on shared responsibility | Reviews or co-maintains team materials. |
| Students | View or submit only | Uses current directions and submits evidence. |
| Department team | View or comment on shared templates | Reuses common resources without damaging active files. |
| Public or external viewers | Usually none in OTS-201 | Public publishing belongs in later courses. |

## Classroom Example

Maya maps permissions before sharing links.

| Folder or file | Access decision |
| --- | --- |
| `Unit 03 - Sensors - ACTIVE` | Maya edits; teammate comments; students do not open folder directly. |
| `Student Links - Unit 03` | Students view linked Docs/Slides and submit Forms. |
| `Teacher Planning - Unit 03` | Maya edits; teammate comments; students no access. |
| `Evidence - Unit 03` | Maya edits; teammate comments; students no access to response Sheets. |
| `Robotics Team Shared Templates` | Team edits template masters; Maya copies into current unit before teaching. |

Common risk:

| Risk | Safer pattern |
| --- | --- |
| Sharing the whole unit folder with students | Share only the current student-facing Doc, Slide deck, Form, or assignment link. |
| Letting students edit directions | Use view-only directions and collect work elsewhere. |
| Storing answer keys beside student links | Keep answer keys in teacher planning or assessment folders. |

## Verification Checklist

- Students can access current materials without seeing teacher-only folders.
- Response Sheets and assessment evidence are protected.
- Teammate access matches actual responsibility.
- Shared templates are separated from active lesson copies.
- Public or external access is not enabled by accident.
- The map names who owns permission review.

## Reflection

Which folder would cause the most damage if shared too broadly? Mark that folder with the clearest access boundary.
