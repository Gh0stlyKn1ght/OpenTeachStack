# Curriculum Tracker

## Why This Matters

The artifact checkpoint is the point where the sheet becomes a living operations system instead of a static list.

For the robotics course, your tracker should tell a substitute exactly what to do and what is safe to leave untouched.

## Core Idea

Build a recurring tracker table that answers substitute-readiness in one pass.

Minimum required row categories:

- Active
- Ready
- Review needed
- Revise
- Archived

## Artifact Build

Create the tracker with these concrete columns:

| Unit | Lesson | Readiness status | Standard | Evidence destination | Owner | Substitution notes | Next review | Blocker |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
| Unit 03 | Sensor launch | Active | Measurement interpretation | `Unit03/Responses` | Maya | Use v1.1 Doc and deck | Fri 2:00 PM | None |
| Unit 03 | Threshold decision | Ready | Data-based reasoning | `Unit03/Reflection` | Maya | No changes to Sheet headers | Fri 10:00 AM | None |
| Unit 03 | Robot recap | Review needed | Communication of evidence | `Unit03/Reflection` | Jordan | Rework intro sentence examples | Wed 3:00 PM | Wording unclear |
| Unit 04 | Path planning | Draft | Design constraints | Not yet set | Jordan | Add path plan links | Thu 4:00 PM | Link missing |
| Unit 04 | Safety walkthrough | Revise | Safety and control | `Unit04/SafetyEvidence` | Maya | Add support script for units | Thu 4:30 PM | None |
| Archive | Unit 02 baseline | Archived | Baseline alignment | Archive folder | Team | Keep read-only | N/A | None |

## Status Rules

| Status | Meaning | Required checks before launch |
| --- | --- | --- |
| Active | Teaching this week | Verified links, owner set, evidence path live |
| Ready | Prepared but not yet launched | Readiness note and blocker field empty |
| Review needed | Evidence or substitutions incomplete | Specific blocker and fix owner set |
| Revise | Needs update before reuse | Planned change and next review date set |
| Archived | Historical only | Removed from student navigation |

## Quality Check

Run this substitute test from your own sheet:

1. Open one Active row.
2. Find where to launch today’s lesson from one link.
3. Find evidence destination without asking the teacher.
4. Find what must be edited before next run.

If any answer requires narrative memory, add that requirement as a table value.

Which status category needs tighter criteria in your current tracker? Tighten one status definition now.
