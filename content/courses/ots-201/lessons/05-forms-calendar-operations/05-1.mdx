# Forms as Checks for Understanding

## Why This Matters

A check-for-understanding Form is not useful because it collects answers. It is useful when the answers change what the teacher does next.

The danger is false efficiency: students submit the Form, the response Sheet fills up, and the teacher never turns the pattern into grouping, feedback, reteaching, or revision.

A workspace system works when the response pattern points to a scheduled teacher action.

## Core Idea

Write the action rule before students answer.

For a quick check, the teacher should know:

- which concept is being checked,
- what answer pattern counts as ready,
- what answer pattern triggers reteaching,
- what answer pattern triggers a small-group check-in,
- where the response Sheet lives,
- when the teacher will review it.

## Classroom Example

Maya wants to know whether students can read ultrasonic sensor data and choose a robot stop threshold. She builds a three-question Form.

Model questions:

| Form question | Why it exists |
| --- | --- |
| Your robot recorded distances of 18 cm, 21 cm, 20 cm, 19 cm, and 22 cm. What stop threshold would you test first? | Checks whether the student can use data instead of guessing. |
| Which reading made you most confident or least confident? Explain in one sentence. | Reveals reasoning and uncertainty. |
| What should your team test again before changing the robot code? | Connects data to the next engineering action. |

Action rules:

| Response pattern | Teacher action |
| --- | --- |
| Most students choose a threshold between 18-22 cm and explain the data. | Move to the next lab challenge. |
| Many students choose 5 cm or 50 cm without using the readings. | Start tomorrow with a two-minute data-range mini lesson. |
| Several teams say they need to retest because readings varied. | Give those teams the retest protocol first. |
| A student reports confusion or access trouble. | Check in before the next build block. |

## Verification Checklist

- Each Form question checks one specific learning decision.
- The questions avoid collecting private information that is not needed.
- The response Sheet name connects to the unit and purpose.
- The review time is scheduled or ready to schedule.
- At least one action rule changes tomorrow's teaching.
- A colleague could understand what to do with the responses.
