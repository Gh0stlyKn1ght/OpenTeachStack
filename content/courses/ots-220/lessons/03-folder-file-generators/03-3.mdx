# Workshop: Generate Unit Folders

## Why This Matters

This workshop converts concept into a repeatable action: open a sheet, run one menu action, and verify the exact generated structure.

## Core Idea

Build a safe workflow:

1. choose one sandbox row
2. run generator
3. verify logs and folders
4. rollback if needed

## Workshop Script

```javascript
function onOpen() {
  SpreadsheetApp.getUi()
    .createMenu("Teacher Automation")
    .addItem("Generate unit folders", "runUnitFolderWorkshop")
    .addToUi();
}

function runUnitFolderWorkshop() {
  const config = getSampleUnitConfig();
  const folderName = buildUnitFolderName(config.unitName, config.sectionName);
  const root = DriveApp.getFolderById(SANDBOX_ROOT_ID);
  const unitFolderId = folderExists(root, folderName)?.getId() || root.createFolder(folderName).getId();
  const unitFolder = DriveApp.getFolderById(unitFolderId);
  const evidence = unitFolder.getFoldersByName("Evidence");
  if (!evidence.hasNext()) unitFolder.createFolder("Evidence");

  const notes = unitFolder.createFolder("Setup Notes");
  const result = {
    level: "info",
    unitName: config.unitName,
    sectionName: config.sectionName,
    unitFolderId,
    notesFolderId: notes.getId(),
    status: "workshop-complete",
  };
  Logger.log(JSON.stringify(result));
  return result;
}
```

Note: this workshop keeps all writes inside sandbox by design.

## Expected Output

```json
{
  "level": "info",
  "status": "workshop-complete",
  "unitName": "Unit 03 - Sensor Lab",
  "sectionName": "Sample Robotics 8A"
}
```

## Failure Drill

If `notesFolderId` is undefined:

- confirm the folder creation line executed
- run the function again with a single clean root
- delete malformed output first

## Quality Check

Run the action from the menu and verify:

- menu appears and runs
- output structure is complete
- no production folder was changed
