# Workshop: Generate Docs from Rows

## Why This Matters

This is the classroom-ready action loop: run from a row, create docs, verify outputs, cleanup, and rerun.

## Core Idea

Build one function that calls:

1. source validation
2. doc copy
3. placeholder fill
4. logging

## Workshop Script

```javascript
const TEMPLATE_DOC_ID = "SANDBOX_TEMPLATE_DOC_ID";

function runDocGeneratorWorkshop() {
  const row = validateSourceRow(getSanitizedSourceRow());
  const template = DriveApp.getFileById(TEMPLATE_DOC_ID);
  const copy = template.makeCopy(`${row.unitName} - Setup`);
  const doc = DocumentApp.openById(copy.getId());

  applyPlaceholdersToDoc(copy.getId(), row);

  const result = {
    level: "info",
    unitName: row.unitName,
    sectionName: row.sectionName,
    docId: copy.getId(),
    status: "doc-generated",
  };
  Logger.log(JSON.stringify(result));
  return result;
}
```

## Setup Notes

- Use sandbox template only.
- Keep the output docs in sandbox root.
- Use sample row only for this workshop.

## Expected Output

```json
{
  "level": "info",
  "unitName": "Unit 03 - Sensor Lab",
  "sectionName": "Sample Robotics 8A",
  "status": "doc-generated"
}
```

## Failure and Recovery

Common failure:

- `Unresolved placeholders` → fix template tokens first
- `Missing required field(s)` → fix row then rerun

After a failure:

- delete partial copy if created
- rerun with fixed row

## Quality Check

Can a teammate run the function and verify the generated doc contains all fields with no placeholders left?
