# Final Build

## Why This Matters

This is the integration pass. You now connect:
- manifest,
- review checklist,
- handoff notes,
- safety checks,
- and manual send controls into one coherent toolkit.

## Core Design

Build a “safe mode” and a “runner” so any run is intentional.

```javascript
const CFG = {
  stop: "STOP_AUTOMATION",
  sheetName: "ReminderQueue",
  statusColumn: 6,
  reviewerColumn: 8,
  notesColumn: 9
};

function buildToolkitBootstrap() {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const cfgSheet = ss.getSheetByName("Config") || ss.insertSheet("Config");
  if (cfgSheet.getLastRow() === 0) {
    cfgSheet.appendRow(["name", "value"]);
    cfgSheet.appendRow([CFG.stop, "FALSE"]);
  }
  const manifest = ss.getSheetByName("AutomationManifest") ||
    ss.insertSheet("AutomationManifest");
  if (manifest.getLastRow() === 0) {
    manifest.appendRow(["component", "owner", "trigger", "status", "lastReviewed"]);
  }
  if (!ss.getSheetByName("AutomationChecklist")) ss.insertSheet("AutomationChecklist");
  if (!ss.getSheetByName("AutomationRunbook")) ss.insertSheet("AutomationRunbook");
  Logger.log("Toolkit bootstrap complete");
}

function isAutomationStopped() {
  const cfg = SpreadsheetApp.getActiveSpreadsheet().getSheetByName("Config");
  const values = cfg.getDataRange().getValues();
  const stopRow = values.find((r) => r[0] === CFG.stop);
  return String(stopRow?.[1]).toUpperCase() === "TRUE";
}

function runApprovedReminderBatch(limit = 10) {
  if (isAutomationStopped()) {
    throw new Error("Automation is stopped by Config STOP_AUTOMATION.");
  }

  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(CFG.sheetName);
  const all = sheet.getDataRange().getValues();
  let sent = 0;

  for (let i = 1; i < all.length && sent < limit; i++) {
    const row = all[i];
    if (row[CFG.statusColumn - 1] !== "approved") continue;

    try {
      MailApp.sendEmail({
        to: row[2],
        subject: `Reminder: ${row[3]}`,
        body: `Hi ${row[1]}, reminder: ${row[3]} is due ${row[4]}.`
      });
      sheet.getRange(i + 1, CFG.statusColumn).setValue("sent");
      sheet.getRange(i + 1, CFG.reviewerColumn).setValue(Session.getActiveUser().getEmail());
      sheet.getRange(i + 1, CFG.notesColumn).setValue(`sent-${new Date()}`);
      sent++;
    } catch (err) {
      sheet.getRange(i + 1, CFG.statusColumn).setValue("error");
      sheet.getRange(i + 1, CFG.notesColumn).setValue(`error:${err.message}`);
    }
  }
  Logger.log(`runApprovedReminderBatch complete: ${sent}`);
}
```

## Delivery Checklist (Build Output)

- `buildToolkitBootstrap()` creates required sheets.
- `runApprovedReminderBatch()` refuses to run when paused.
- successful sends set `status = sent`.
- failures are written into notes and status.

## Manual Safety Drill

1. Set `STOP_AUTOMATION = TRUE`, run `runApprovedReminderBatch()`, confirm throw.
2. Set `STOP_AUTOMATION = FALSE`.
3. Run again with one approved row and one draft row.
4. Confirm only approved row sends.
