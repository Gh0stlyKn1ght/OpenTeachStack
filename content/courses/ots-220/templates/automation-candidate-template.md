# OTS-220 Automation Candidate Template

Use this template for Chapter 01.

- Workflow: _What repetitive task are you choosing to automate?_
- Why now: _What cost of time or inconsistency does this remove?_
- Safety boundary: _What will not be automated and why?_
- Trigger condition: _When should automation run and when should it stop?_
- Rollback condition: _What is the first step if output looks wrong?_
- Dry run test: _What does one successful test look like?_
