# OTS-240 Examples

Use these examples to anchor open-resource and GitHub coaching.

- completed-artifact.md: curriculum repo README and source notes.
- weak-version.md: missing attribution or license coverage.
- strong-version.md: complete attribution, licensing, and excluded-media notes.
- reviewer-notes.md: contribution and copyright review checklist.

Assumptions:
- Replace placeholders for repo URLs with the current cohort repo.
- Do not include proprietary data or actual student submissions.
