# Strong Version

## Strong Open Curriculum Artifact

| Artifact | Strong evidence |
| --- | --- |
| README | Purpose, audience, contents, privacy boundary, reuse note, and maintenance rule are visible. |
| Source notes | Official, OER, blog, and excluded sources are labeled with classroom use decisions. |
| License notes | Original materials, third-party terms, and excluded materials are separated. |
| Contribution checklist | Review rules include source, license, privacy, accessibility, and maintenance. |
| Release checklist | Publish/internal/revise/block decision is tied to evidence. |

## Strong Review Pattern

- **Ready:** another educator can reuse the repository without guessing source, license, or privacy boundaries.
- **Revise:** one source, license, accessibility, or maintenance note is incomplete.
- **Blocked:** repository includes restricted files, real student work, private links, or unlicensed media.
