# License and Terms Decisions

## Why This Matters

“Public” does not always mean “free to reuse”.
You need clear terms for each file.

## Working with License Types

Start with these three for your course repo:

- `CC BY 4.0` — remix allowed, attribution required.
- `CC BY-SA 4.0` — remix allowed, derivative work must stay same license.
- `All Rights Reserved` — no remix without written permission.

## Mini Exercise

Create a mapping for three artifacts.

| Artifact | License candidate | Allowed action |
| --- | --- | --- |
| Lesson plan draft | CC BY 4.0 | share and remix |
| Student sample | All Rights Reserved | private classroom use only |
| Photo from robotics club | CC BY-SA 4.0 | remix with same license |

## Safety Rule

Any file derived from another source must keep the most restrictive term unless the source explicitly allows broader use.

## Concrete Command

```text
# record this in your log as source + term per artifact
Artifact | Original Source | License | Attribution Name
```

## Quality Check

You can only move forward if each artifact has a chosen license term and an owner note explaining any restriction.
