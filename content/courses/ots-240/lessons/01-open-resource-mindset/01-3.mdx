# Open Resource Decision Log

## Build Task

Create a working artifact called **Open resource decision log**.
It becomes the source of truth for sharing decisions.

## Required Columns

Use these columns in a sheet, markdown table, or issue list:

- artifact
- source
- license
- allowed use
- attribution text
- privacy constraint
- owner
- review date
- status

## Starter Template

```text
| artifact | source | license | allowed use | attribution | privacy rule | owner | review date | status |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
| Robot starter packet | Internal teaching materials | CC BY 4.0 | remix + redistribute | "Open Teacher Toolkit" | no student data | You | 2026-09-20 | ready |
| Sample student reflection | internal classwork | All Rights Reserved | private use only | none | remove student names | You | 2026-09-20 | blocked |
```

## Practical Steps

1. Convert two existing generic lesson files into real rows.
2. Add one attribution text per external source.
3. Add `privacy rule` for every row that contains learning evidence.

Keep this log open to review during every chapter checkpoint in OTS-240.
