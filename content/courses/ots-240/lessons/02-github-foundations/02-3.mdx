# Workshop: Create a Curriculum Repo

## Task Goal

Create or verify a repository with required teaching folders and a clear first commit.

## Checklist

1. Create repo with this structure: `content/`, `docs/`, `LICENSE`, `README.md`.
2. Add initial files from your open resource decision log.
3. Commit with a message following your course pattern.

## Practical Script (Low Risk)

```bash
git init

git add README.md LICENSE content
git commit -m "course: ots-240 | scaffold | create teaching repository structure"
```

## Verification

- one branch exists for work (`week-xx`),
- `README.md` explains setup in fewer than 6 bullet points,
- at least one commit shows ownership and reason.
