# Checkpoint Review

## Review Standard

- **Ready**: repo has stable folder map, documented workflow, and branch name convention.
- **Revise**: README missing setup, branch and commit convention.
- **Blocked**: no version history yet (no initial commit).

## Required Evidence

- one branch name reflecting current task,
- one commit with clear reason,
- README has setup and review notes,
- no student personal data in tracked files.
