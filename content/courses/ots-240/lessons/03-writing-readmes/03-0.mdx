# Chapter Plan

## Why This Matters

Your README is often the first file a substitute or partner teacher opens.
If it is unclear, they cannot start quickly.

## Target

Create a README that helps people: understand the course, use it safely, and contribute safely.

## Front Matter for the README Project

Before writing sections, set this plan:

- primary audience: team teachers,
- level: intermediate,
- expected toolchain: markdown + GitHub web editor,
- safety rule: student data excluded.

## Draft Sections

- What this course covers
- Prerequisites
- Setup steps
- Contribution flow
- Attribution and licensing note
