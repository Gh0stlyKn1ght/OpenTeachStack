# Build Task: Course README

## Objective

Write a practical `README.md` draft that supports a substitute, a parent, and a peer reviewer.

## Content Checklist

- Course purpose in one paragraph.
- Folder map.
- Setup flow.
- Link to license and contribution notes.
- Last update date.

## Sample Mini-README

```markdown
# Robotics Systems: Open Resource Unit

This course teaches classroom workflows for safe sharing, versioning, and lesson assembly.

## Quick Start
1. Clone repository
2. Open `content/` and read chapter index
3. Review `LICENSE` before sharing

## Contribution
- Branch naming: `week-XX-description`
- Use clear commit messages
- Do not commit student names or raw submissions
```

## Evidence

Save your draft in `README.md` and note one reviewer comment to improve clarity.
