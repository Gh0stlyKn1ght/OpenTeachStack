# Checkpoint Review

## Rubric

- **Ready**: README includes audience, setup, safety, and license sections.
- **Revise**: setup lacks sequence or no owner/contact.
- **Blocked**: includes student-specific private details.

## Required Evidence

- at least one pull request preview or local file diff showing README changes,
- a person from another team can summarize next steps from it,
- license link present.
