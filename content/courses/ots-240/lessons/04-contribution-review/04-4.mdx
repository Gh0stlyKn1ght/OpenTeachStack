# Checkpoint Review

## Chapter Criteria

- **Ready**: PR template exists, contribution rules documented, one reviewed sample logged.
- **Revise**: no checkboxes, vague decisions, missing reviewer field.
- **Blocked**: no review before publish.

## Evidence to keep

- one issue entry,
- one pull request draft,
- one review decision logged.
