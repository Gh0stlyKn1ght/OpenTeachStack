# Chapter Plan

## Why This Matters

Publishing is not only making files public.
It is making ownership, limits, and credit explicit.

## Chapter Output

You will complete a publish bundle:
- `SOURCE_NOTES.md`
- `ATTRIBUTION.md`
- `LICENSE` entries aligned with course rules

## Setup Step

Create an attribution plan now:
- which assets are your original work,
- which are adapted,
- which are disallowed for reuse.
