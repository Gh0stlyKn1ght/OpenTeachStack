# Build Task: Attribution and License File

## Objective

Create and verify final attribution files for publishing.

## Required Files

- `LICENSE`
- `SOURCE_NOTES.md`
- `ATTRIBUTION.md`

## Example `SOURCE_NOTES.md`

```markdown
# Source Notes
- `images/robot-arm.png` from RoboticsLab open pack (CC BY 4.0)
- `lessons/week-1.mdx` authored by You
```

## Acceptance

- Every external resource appears in `SOURCE_NOTES.md`.
- `LICENSE` matches the chosen project terms.
- Attribution text is understandable without opening PRs.
