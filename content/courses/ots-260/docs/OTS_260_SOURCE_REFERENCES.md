## Course-Writing Rules

- Use official docs for tool behavior; do not rely on memory or old screenshots.
- Treat tool terms as current only after checking the platform source.
- Do not claim AI-generated works are automatically copyright-free or automatically protected.
- Require source notes for borrowed media and disclosure notes for AI-generated media.
- Keep student data and identifiable student media out of examples unless policy and consent explicitly allow it.
