# OTS-260 Examples

Use these examples to ground media and lesson delivery work in classroom-safe production.

- completed-artifact.md: short video plan and media packet draft.
- weak-version.md: image and slide prompts without usage constraints.
- strong-version.md: prompt with usage note plus accessibility checks.
- reviewer-notes.md: caption/transcript/readability verification list.

Assumptions:
- Keep all media samples school-safe and consent-aware.
- Record source and usage constraints for every generated asset.
