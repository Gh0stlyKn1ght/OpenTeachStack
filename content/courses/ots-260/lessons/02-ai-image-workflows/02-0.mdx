# Chapter Plan

## Why This Matters

In this chapter, AI image workflows should be judged by decisions, not aesthetics. We only use an image when it directly helps learners complete a learning action and passes accuracy, bias, and accessibility checks.

## Core Idea

Use the same five-step loop for every image:

1. Define the teaching target.
2. Draft a constrained prompt.
3. Generate several candidates.
4. Review with a scoring rubric.
5. Record the final decision in the artifact log.

Each step must be visible in `Image prompt and review log` so another educator can replicate the same choice.

## Chapter Outcomes

| Outcome | How to prove it |
| --- | --- |
| Clear image purpose | A row states exactly what confusion the image addresses |
| Reproducible prompting | Prompt text includes objective, style, constraints, safety rules |
| Reviewed quality | Pass/fail notes on accuracy, readability, and bias are present |
| Responsible publishing | `READY / REVISE / BLOCKED` is set with owner + recheck date |

## Chapter Workflow Template

Use this template when filling the chapter row before writing any visuals.

| Field | Required entry |
| --- | --- |
| Lesson focus | e.g., “Explain branching logic with one conditional diagram” |
| Student action | What learners should do with the final image |
| Prompt constraints | No medical/legal claims, no stereotypes, no logo, no real people unless consented |
| Accessibility check | Minimum contrast, readable labels, alt-text draft |
| Ownership | Who can approve final publish |
| Status | READY, REVISE, or BLOCKED |
| Recheck trigger | Date or condition that prompts another review |

## Classroom Example

Context: A teacher needs one visual to introduce loops in computer science.

| Field | Example |
| --- | --- |
| Lesson focus | “Show loop execution over 3 steps” |
| Student action | Students trace arrows and fill in missing state |
| Prompt constraints | “Vector style, no branding, include arrows and labels, max 3 colors, no copyrighted characters” |
| Accessibility check | Labels at least 18px equivalent, high contrast, alt text drafted |
| Ownership | Lead course designer |
| Status | REVISE → READY |

## Common Mistake

Skipping the recheck date or owner makes the chapter hard to maintain. If no one is accountable, your image can silently drift out of alignment with standards.

## Quality Check

Move on only when your artifact row has:
- a teaching objective,
- prompt constraints,
- at least one review note,
- and a decision with owner/date.
