# Mermaid and Simple Diagram Tools

## Why This Matters

Text-based diagrams are review-friendly. Mermaid or equivalent text tools make visuals editable, searchable, and versioned without relying on binary-only memory.

## Core Idea

Start with a minimal valid diagram, then add constraints for clarity.

```text
flowchart TD
  A[Start] --> B[Input]
  B --> C{Check valid?}
  C -- Yes --> D[Process]
  C -- No --> E[Reject]
  D --> F[Output]
  E --> F
```

## Tooling Checklist

- Keep labels short and explicit.
- Keep one concept per diagram.
- Use edge labels like `if`, `then`, `next`, not decorative words.
- Add color only after logic is correct.

## Common Mistake

Focusing on aesthetics before correctness. A polished diagram with wrong labels is still wrong.

## Reflection

What is the single line in your diagram that causes the most confusion for students? Make it explicit.
