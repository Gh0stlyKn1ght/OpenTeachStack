# When Video Helps

## Why This Matters

Video is useful when repetition, pacing, or demonstration is more efficient than repeated live teaching.

## Core Idea

Choose video only when one of these is true:
- Students need a missing-step replay.
- Procedure is too fast to demonstrate live consistently.
- Remote access is required.

If none apply, keep audio/text-only.

## Decision Rubric

| Scenario | Use video? | Why |
| --- | --- | --- |
| Process walkthrough | Yes | Reduces repeated confusion |
| Complex reasoning debate | No | Better with discussion forums |
| Homework troubleshooting | Yes | Supports asynchronous review |

## Practice

Create a one-page decision log with 3 scenarios from your course and mark each `YES` or `NO`.

## Common Mistake

Using video for every topic. Unnecessary video increases load and maintenance burden.

## Reflection

Which topic would improve most if moved from video to guided text?
