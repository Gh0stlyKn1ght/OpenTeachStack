# Scripts, Captions, and Transcripts

## Why This Matters

Accessible media is not optional. Captions and transcripts are learning infrastructure, not polish.

## Core Idea

Draft a simple script block before recording:

- Hook (5 seconds)
- Learning objective (10 seconds)
- 2–4 steps
- Exit checkpoint

After recording, produce:
- Accurate caption file
- Searchable transcript
- One accessibility note for names, symbols, and on-screen references.

## Script Template

`[Time] [On-screen action] [Voice line] [Visual note]`

## Practice

Complete one script + one caption snippet + one transcript snippet in `Short video plan`.

## Common Mistake

Skipping transcript because caption timing is "good enough." transcripts are needed for indexing and rereads.

## Reflection

What line in your script is hardest to caption clearly and why?
