# Short Video Plan

## Why This Matters

The artifact is the real deliverable: one reusable plan others can adopt and review quickly.

## Core Idea

Create a production-ready plan with:
- script version,
- recording settings,
- accessibility attachments,
- review state.

## Build Grid

| Item | Value |
| --- | --- |
| Video title | [title] |
| Length target | [00:xx] |
| Script version | [v1 / v2] |
| Recording settings | [tool + format + resolution] |
| Caption status | draft/reviewed/published |
| Transcript status | missing/reviewed/final |
| Reviewer | [name] |
| Final status | READY / REVISE / BLOCKED |

## Practice

Fill the grid and add two concrete review notes:
- one for content clarity,
- one for accessibility.

## Common Mistake

Tracking only production date and forgetting versioning for edits. Without versioning, review history is lost.

## Reflection

If this plan is reused next month, what will immediately be outdated?
