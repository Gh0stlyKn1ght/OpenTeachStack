# Checkpoint Review

## Why This Matters

The review gate ensures short videos are usable and accessible before release.

## Core Criteria

- Concept coverage is accurate
- Duration stays within planned range
- Captions and transcript are present
- Action instructions are clear
- No unresolved accessibility gaps

## Gate

| Status | Requirement |
| --- | --- |
| READY | Script and recording accepted; captions/transcript complete |
| REVISE | One narrow issue that can be fixed in one version |
| BLOCKED | Accuracy or accessibility risk remains |

## Review Task

Add one final row with:
- total clips,
- ready/revise/blocked count,
- reviewer,
- recheck date.
