# Chapter Plan

## Why This Matters

This chapter is the capstone of media delivery: if your materials are not accessible, the lesson is not inclusive.

## Core Idea

An accessible packet contains the same core content as your media set, plus clear adaptations.

- visual alternatives
- captions/transcripts
- text navigation cues
- review ownership

## Outcomes

| Outcome | Proof |
| --- | --- |
| Accessibility checklist in place | Completed checklist row |
| Alternate representations ready | Alt text + transcript plan |
| Review loop defined | Owner + status + recheck date |
| Revision logic | READY/REVISE/BLOCKED criteria |

## Plan First Row

Fill these fields in `Accessible lesson media packet`:
- target learner needs,
- planned adaptations,
- final owner.

## Reflection
