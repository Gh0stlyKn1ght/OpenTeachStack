# Media Packet Checklist

## Why This Matters

A checklist prevents accessibility from being forgotten after creation and before publishing.

## Core Checklist

- Alt text for each image/diagram
- Captions for each short video
- Logical heading order in slide decks
- High-contrast text/background
- Keyboard-accessible activity links
- Transcript for key videos

## Build Task

Create `Accessible lesson media packet` with a row per asset type:

| Asset | Accessibility check | Status | Notes |
| --- | --- | --- | --- |
| Image | Alt text present | READY/REVISE/BLOCKED | [notes] |
| Video | Captions + transcript | READY/REVISE/BLOCKED | [notes] |
