# Strong Version

## Strong Teacher Safety Artifact

| Artifact | Strong evidence |
| --- | --- |
| Personal risk map | Names exposure, calm action, evidence, and review trigger without storing secrets. |
| Password manager and MFA checklist | Records account status without passwords, backup codes, or screenshots. |
| Phishing response checklist | Gives pause, inspect, verify, report, recover steps. |
| Public surface review | Separates public-safe examples from private student or school-only information. |
| Incident response plan | Names first step, evidence, contact path, and follow-up. |

## Strong Review Pattern

- **Ready:** lowers a named risk and avoids collecting unnecessary sensitive data.
- **Revise:** the action is useful but missing evidence, contact path, or review trigger.
- **Blocked:** the artifact stores secrets, exposes private student data, or relies on unapproved reporting paths.
