# Public Profile Audit

## Why This Matters

Teacher safety work should reduce exposure when a teacher can point to the decision, the classroom use, the evidence, and the next review step.

In **Social Media and Doxxing Risk**, the work sits inside teacher digital safety, account hygiene, public profile review, connection safety, and incident response. The risk is turning safety into fear instead of reducing specific exposure with practical routines. This lesson narrows that risk into one visible update to **Public profile audit**.

## Core Idea

Build the **Public profile audit** as a usable artifact, not just a note. The finished version shows the decision, the classroom context, the evidence, and the next maintenance step.

The goal is not to make a perfect document in one pass. The goal is to make the teacher decision clear enough that another educator can inspect it, reuse it, or challenge it without needing a private explanation.

## Lesson Focus

Because this is an artifact lesson, the teacher turns public profile audit into something another educator can use. The artifact needs labels, examples, status, review notes, and a next action instead of a vague promise to improve later.

In 05.5, focus on public profile audit. Choose one concrete classroom material or teacher decision from this list: an account, profile, device setting, website form, repository, domain record, or incident note. Connect it to public profile audit, then decide whether it needs to be created, revised, tested, credited, restricted, published, or archived.

Connect public profile audit back to social media doxxing risk. That connection matters because a polished artifact can still fail if the lesson purpose, boundary, evidence, and maintenance routine are not visible.

## Classroom Example

Use this working situation: a teacher is auditing accounts, public profiles, device settings, repo exposure, website forms, and response steps without collecting private student information.

## Quality Check

Before moving on, test the work against this standard: the action lowers a named risk, avoids unnecessary data collection, and gives the teacher a calm next step.

A reviewer should be able to find the public profile audit decision, inspect the public profile audit evidence, and understand how it supports social media doxxing risk.

If the work only names a tool or a file, revise it. The artifact needs to show the teaching purpose, evidence, boundary, and maintenance routine.
