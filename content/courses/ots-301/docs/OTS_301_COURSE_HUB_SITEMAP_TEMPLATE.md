## Navigation Rules

- Keep the top navigation to the pages visitors need most.
- Do not link public pages to private drafts, gradebooks, class rosters, or student files.
- Put source notes and update dates where visitors can find them.
- Archive or redirect old materials instead of leaving stale pages in the main path.

## Launch Readiness

- [ ] Each planned page has a purpose and owner.
- [ ] Every public/private boundary is documented.
- [ ] Source notes and update notes have a planned location.
- [ ] Old or seasonal materials have an archive plan.
- [ ] A reviewer can find the main course materials in two clicks or fewer.
