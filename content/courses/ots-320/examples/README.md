# OTS-320 Examples

Use these examples to train safe AI agent collaboration.

- completed-artifact.md: safe prompt and allowed/blocked file matrix.
- weak-version.md: overbroad prompt and missing diff review.
- strong-version.md: scoped prompt with guardrails and verification steps.
- reviewer-notes.md: diff, tests, and rollback review checklist.

Assumptions:
- No real secrets or protected data in prompts.
- All change samples use fake repos and synthetic test inputs.
