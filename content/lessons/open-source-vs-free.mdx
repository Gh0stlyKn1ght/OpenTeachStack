# Open Source vs Free

"Free" and "open source" are not the same thing. Confusing them leads teachers to make dependency decisions they later regret.

## Free (as in beer)

A tool that is free costs nothing to use. That is all it guarantees.

- **Google Classroom** — Free, but Google controls every aspect of the product.
- **Canva for Education** — Free, but Canva decides what features teachers get.
- **Kahoot** — Free tier exists, but the best features require a paid plan.

Free tools can:
- Change pricing at any time
- Remove features you depend on
- Shut down entirely
- Lock your data in proprietary formats
- Sell your usage data

Free means you do not pay money. It does not mean you are not paying.

## Open Source (as in freedom)

An open-source tool makes its source code publicly available under a license that grants specific freedoms:

1. **Freedom to use** — Run it for any purpose
2. **Freedom to study** — Read and understand the code
3. **Freedom to modify** — Change it to fit your needs
4. **Freedom to distribute** — Share your modifications

Open-source tools can still cost money (for hosting, support, or services), but the code itself is accessible and reusable.

Examples:
- **Next.js** — MIT license. The framework this site is built with.
- **LibreOffice** — Open alternative to Microsoft Office.
- **Moodle** — Open-source LMS used by universities worldwide.
- **WordPress** — Open-source CMS powering 40%+ of the web.

## Why This Matters for Teachers

When you build your curriculum system on open-source tools:

- **You can leave.** Your content is in standard formats (Markdown, HTML) that any platform reads.
- **You can customize.** If a tool does not do exactly what you need, you (or someone you hire) can change it.
- **The tool survives the company.** If the company behind an open-source project closes, the code lives on. The community can fork and continue it.
- **Your students learn real tools.** Open-source tools are what professionals use. Git, Linux, VS Code — all open source.

<RealityCheck>
Open source does not mean "better" in every case. Google Docs is a better document editor than most open-source alternatives. The point is not to replace everything with open source. The point is to understand what you are giving up when you choose a closed tool.
</RealityCheck>

## Creative Commons for Content

Open source is for code. For content (lessons, images, curriculum materials), the equivalent is **Creative Commons**.

The most common CC licenses for education:

| License | Allows | Requires |
|---------|--------|----------|
| CC BY | Any use | Attribution |
| CC BY-SA | Any use | Attribution + ShareAlike |
| CC BY-NC | Non-commercial use | Attribution |
| CC BY-NC-SA | Non-commercial use | Attribution + ShareAlike |

This course uses **CC BY-NC-SA 4.0** for content: you can use, adapt, and share the materials for non-commercial purposes, as long as you credit the source and share your adaptations under the same license.

<TeacherNote>
Understanding licensing is a professional skill. When teachers use images, texts, or resources without understanding licenses, they model poor digital citizenship. When they understand and apply Creative Commons, they model intellectual honesty and contribute to the commons.
</TeacherNote>
