# OTS-280 Private-Use Templates

Date: 2026-06-22

Status: active private-use checklist set

These templates support OTS-280 without collecting sensitive values in the repo. They are patterns teachers copy into a private notebook, local file, district-approved document, or password-manager notes area. Do not submit completed private versions as course evidence.

## Template Safety Rules

- Use labels such as `school email`, `personal email`, `course site`, or `public profile`; do not write full usernames, passwords, recovery emails, backup codes, home addresses, student names, or private URLs.
- Share only sanitized summaries for peer review.
- Keep private versions out of public repositories, LMS discussion posts, AI tools, and shared class folders.
- If a template reveals an active incident, stop the course activity and use the escalation path.
