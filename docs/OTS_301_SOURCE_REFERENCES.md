Date: 2026-06-22

Status: active source queue

OTS-301 should use official platform documentation and standards references whenever web behavior, deployment, DNS, accessibility, or publishing guidance matters.

## Course-Writing Rules

- Use official host documentation for deployment behavior, not old screenshots.
- Use reserved placeholder domains for public examples.
- Treat DNS tokens, billing details, account screenshots, and verification records as private.
- Require source notes, update dates, HTTPS checks, and broken-link checks before public launch.
- Review accessibility, mobile readability, and privacy before sharing a live site.
