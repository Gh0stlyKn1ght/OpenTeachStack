# OpenTeachStack Full Pathway Release

The full OpenTeachStack pathway is now live in authored form.

This milestone moves beyond OTS-101 foundations and includes all nine pathway courses:

- OTS-101 Teaching Teachers Foundations
- OTS-201 Google Workspace Systems for Teachers
- OTS-220 Apps Script for Teacher Automation
- OTS-240 Open Resources & GitHub for Educators
- OTS-260 AI Media & Lesson Delivery
- OTS-280 Cyber Safety for Educators
- OTS-301 Teacher Course Sites
- OTS-320 AI Coding Agents for Educators
- OTS-399 Capstone Studio

Across the pathway, route-visible books, chapter systems, source-linked artifacts, and
content safety reviews are now in place. The release assets are organized for practical
teaching use:

- Course-owned lesson content under `content/courses/{course}`
- Course-specific templates, artifacts, references, and reusable checklists
- Evidence and traceability surfaces (`/evidence`, `/skills`, `/kb/source-bank`)
- Verified route checks and linting as part of release reporting

Repository-wide evidence is tracked in:

- `docs/BUILD_VERIFICATION.md`
- `docs/KB_ARCHITECTURE.md`
- `docs/BUILD_VERIFICATION.md` (latest validation snapshot)

We also have:

- Source-backed release evidence in `/evidence`
- Community contribution guidance in `/contribute`
- Brand standards applied consistently across README, templates, and routing structure
- Shared course metadata and cross-course traceability tooling

If you are coming to this as a new pathway teacher, start with:

- OTS-101: `/book/ots-101`
- Course cards and pathways: `/`
- Knowledge Base: `/kb`
- Knowledge Base library: `/kb/library`

The pathway follows the same safety and quality rhythm throughout:

Source -> Prompt -> Build -> Verify -> Teach -> Archive -> Improve

Release date: 2026-06-22
