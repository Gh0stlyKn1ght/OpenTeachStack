# OTS-101 Lesson Type Audit

Date: 2026-06-26

## Core finding

OTS-101 now has real lesson bodies, but many lessons still wear the same artifact-lesson uniform. The fix is not less structure. The fix is the right structure for the lesson purpose.

> Every lesson needs instructional structure, but not every lesson needs the same structure.

## Lesson type counts

- Artifact Build Lesson: 10
- Capstone Assembly Lesson: 3
- Case Study / Mistake Log: 4
- Checkpoint / Review Lesson: 10
- Comparison Lesson: 3
- Concept Lesson: 25
- Framework Lesson: 2
- Workflow Lesson: 3

## Remediation rule

Required course elements can exist across a chapter. They do not all need to appear in every single lesson. Artifact lessons should keep build/check/capstone structure. Checkpoints should act like reviews. Concept lessons should teach mental models. Case studies should teach from what broke.
