# Prompt anatomy

## The real problem

AI output gets weak when the teacher gives a vague task and no boundaries.

## Quick answer

A useful prompt names the role, context, source, task, constraints, output format, and verification rules.

## When to use this

Use this when asking AI to draft lesson materials, rubrics, directions, quiz ideas, or revision feedback.

## Workflow

1. Name the role: teacher assistant, editor, rubric reviewer.
2. Provide classroom context.
3. Give the source or say no source is provided.
4. Define the output.
5. Set constraints.
6. Ask for verification notes.

## What to verify

AI can draft. You still decide what reaches students.

## Common mistakes

- Asking for a full lesson with no standard, source, or audience.
- Letting the AI invent citations.
- Keeping language that does not sound like you.
