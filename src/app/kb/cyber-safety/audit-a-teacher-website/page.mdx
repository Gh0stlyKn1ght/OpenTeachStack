# Audit a teacher website

## The real problem

Teacher sites can accidentally expose old links, private files, student names, or confusing directions.

## Quick answer

Audit public pages for privacy, accessibility, source clarity, update freshness, and broken links.

## When to use this

Use this before sharing a course site, portfolio, resource hub, or lesson page.

## Workflow

1. Open the site in a private browser window.
2. Check every navigation link.
3. Search for student names or private identifiers.
4. Inspect shared file permissions.
5. Review accessibility basics.
6. Add update and source notes where needed.

## What to verify

No student data, no private files, no dead links, and no resources you cannot explain.

## Common mistakes

- Auditing while signed in.
- Forgetting embedded Docs or Slides permissions.
- Leaving old year-specific announcements on public pages.
