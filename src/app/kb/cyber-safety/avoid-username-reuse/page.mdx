# Avoid username reuse

## The real problem

One reused username can connect personal, school, public, and old accounts.

## Quick answer

Use different usernames for personal, professional, and experimental work.

## When to use this

Use this before creating public profiles, GitHub accounts, demo sites, or classroom tool accounts.

## Workflow

1. List current public usernames.
2. Separate personal and educator identities.
3. Choose a professional naming pattern.
4. Avoid using student-facing handles for experiments.
5. Document which accounts are public.

## What to verify

Search the username before reusing it. Check profile photos, bios, links, and old posts.

## Common mistakes

- Reusing a personal gamer handle for a school project.
- Linking every account back to the same profile.
- Forgetting old usernames that still appear in search.
