# Public profile audit

## The real problem

Public profiles often accumulate old links, mixed identities, and details that no longer need to be public.

## Quick answer

Review what a stranger can see, then remove stale, personal, or confusing details.

## When to use this

Use this before publishing a portfolio, GitHub profile, teacher site, or public resource bank.

## Workflow

1. Search your name and usernames.
2. Open profiles while signed out.
3. Check avatar, bio, links, pinned work, and contact options.
4. Remove stale or personal details.
5. Keep a professional profile note.

## What to verify

Make sure the profile matches the identity and audience you intend.

## Common mistakes

- Reviewing only the profile dashboard.
- Keeping old links because they are familiar.
- Mixing personal and school contact paths.
