# Organize a course folder

## The real problem

Drive becomes stressful when drafts, student files, yearly copies, and shared materials all live together.

## Quick answer

Separate source material, teachable copies, student-facing exports, and archive folders.

## When to use this

Use this before building a unit, sharing materials, or asking an automation to create folders.

## Workflow

1. Create one course root folder.
2. Add folders for units, templates, source notes, exports, and archive.
3. Keep editable teacher files away from student-facing exports.
4. Use naming conventions that include course, unit, and year.
5. Review permissions before sharing.

## What to verify

Check ownership, link sharing, student privacy, and whether another teacher could find the material.

## Common mistakes

- Sharing the teacher draft folder.
- Mixing yearly archives into active units.
- Naming files only by vague lesson titles.
