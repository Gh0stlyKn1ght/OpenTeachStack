# Knowledge Base Library

The Library is now part of the Knowledge Base. Use this page when you need a prompt, template, source, checklist, workflow, example, or reusable artifact without deciding which support shelf it belongs to.

## Main shelves

- [Prompt Library](/kb/prompts): copy-ready teacher prompts with verification and safety checks.
- [Template Library](/kb/templates): reusable course artifacts, planning sheets, and downloadable Markdown templates.
- [Source Bank](/kb/source-bank): official references and source-backed links for course decisions.
- [Safety Guide](/kb/safety): privacy, copyright, AI verification, accessibility, and classroom-readiness checks.
- [Workflows](/kb/playbooks/source-prompt-build-verify-teach-archive-improve): source, prompt, build, verify, teach, archive, and improve.
- [Examples](/kb/examples): sample artifacts and finished-enough teaching packets.
- [Field Notes](/kb/field-notes): project notes, lessons learned, and human context.
- [Tools and Resources](/kb/resources): curated tools, official docs, and practical resource links.
- [Open Source and Licensing](/open-source): reuse rules, contribution notes, and project licensing.

## Use it like this

1. Search the Knowledge Base for the thing you need.
2. Open the prompt, template, source, checklist, or workflow.
3. Attach the output to the matching course artifact.
4. Verify safety, sources, accessibility, and classroom fit before using it with students.

## Safety reminder

Do not paste student-identifiable information, private school data, passwords, API keys, tokens, or private links into prompts, templates, examples, or support workflows.
