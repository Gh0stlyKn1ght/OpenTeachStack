# Prompt library

## The real problem

Prompt examples become hard to reuse when they are buried inside lessons.

## Quick answer

Keep prompts in the Library and link them from KB articles and course chapters.

## When to use this

Use this when you need a reusable prompt pattern for lessons, rubrics, directions, source checks, or agent tasks.

## Workflow

1. Pick a prompt pattern.
2. Replace placeholders with real course context.
3. Add sources and constraints.
4. Verify the output before teaching.

## What to verify

The prompt should preserve teacher judgment and avoid invented sources.

## Common mistakes

- Copying a prompt without changing the audience.
- Treating one prompt as universal.
