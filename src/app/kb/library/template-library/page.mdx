# Template library

## The real problem

Templates save time only when teachers understand what to keep, change, and verify.

## Quick answer

Keep templates in the Library. Use KB articles to explain when and how to apply them.

## When to use this

Use this when building lesson pages, source logs, rubrics, folder structures, and safety checks.

## Workflow

1. Pick the smallest template that fits.
2. Remove unused sections.
3. Add real sources and classroom context.
4. Verify before sharing.

## What to verify

Do not let the template overwrite the teacher.

## Common mistakes

- Keeping template language that students will not understand.
- Adding every optional section.
