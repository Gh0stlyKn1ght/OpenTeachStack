# OpenTeachStack Knowledge Base

A field guide for educators entering the tech world.

Courses teach the path. The knowledge base solves the problem in front of you. Start with tomorrow. Build toward a system.

## How to use it

Use the left navigation when you know the category. Use search when you know the problem, prompt, template, checklist, or tool you need.

## Core rule

AI can draft. You still decide what reaches students. If you cannot verify it, do not teach it yet.
