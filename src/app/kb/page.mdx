# OpenTeachStack Knowledge Base

The Knowledge Base is the quick-help layer for OpenTeachStack.

Courses teach the path. The KB solves the problem in front of you: a prompt, source check, safety rule, workflow, template, tool, or troubleshooting move.

## How to use it

Use the left navigation when you know the topic. Use search when you know the problem, prompt, template, checklist, or tool you need.

## Core rule

AI can draft. You still decide what reaches students. If you cannot verify it, do not teach it yet.

## Main tools

- [Prompt Library](/kb/prompts): copy-ready prompts for teacher-owned course artifacts.
- [Template Library](/kb/templates): reusable planning, verification, safety, publishing, and revision templates.
- [Source Bank](/kb/source-bank): verified source links, official documentation, and resource notes.
- [Tools & Platforms](/kb/tools-platforms): software vocabulary, install links, official docs, and setup guidance.
- [Build Notes](/build-notes): what we are learning while building OpenTeachStack.
