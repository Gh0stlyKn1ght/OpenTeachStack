# Playbook: Audit a teacher website

## The real problem

Teacher sites can drift: old links, public files, private data, and unclear directions build up quietly.

## Quick answer

Use the canonical guide: [Audit a teacher website](/kb/cyber-safety/audit-a-teacher-website).

## When to use this

Use this before sharing a teacher-owned site, portfolio, course hub, or lesson page.

## Workflow

Open the site signed out, check links, inspect shared files, search for private identifiers, and review accessibility basics.

## What to verify

No student data, no private files, no dead links, and no mystery sources.

## Common mistakes

- Auditing while signed in.
- Forgetting embedded files and videos.
