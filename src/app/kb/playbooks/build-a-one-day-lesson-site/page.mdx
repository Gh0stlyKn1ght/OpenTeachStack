# Playbook: Build a one-day lesson site

## The real problem

The same workflow belongs in both Lesson Building and Playbooks, but the reusable instructions should live in one canonical place.

## Quick answer

Use the canonical guide: [Build a one-day lesson site](/kb/lesson-building/build-a-one-day-lesson-site).

## When to use this

Use this playbook when you need a compact lesson page for tomorrow, a substitute, or a student-facing workflow.

## Workflow

Follow the Lesson Building guide and keep the artifact small: target, materials, steps, check, sources, and backup plan.

## What to verify

Check links, captions, privacy, accessibility, and whether students can begin without a second explanation.

## Common mistakes

- Turning one day into a giant course site.
- Starting with visual polish before the task is teachable.
