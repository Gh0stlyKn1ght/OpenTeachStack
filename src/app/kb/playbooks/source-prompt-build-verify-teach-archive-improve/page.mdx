# Source, prompt, build, verify, teach, archive, improve

## The real problem

Teachers need a repeatable way to move from idea to teachable artifact without losing judgment or evidence.

## Quick answer

Use the seven-step loop: source, prompt, build, verify, teach, archive, improve.

## When to use this

Use this for lessons, rubrics, course pages, AI drafts, agent edits, and reusable templates.

## Workflow

1. Source: find trustworthy material.
2. Prompt: describe the task and boundaries.
3. Build: create the smallest useful artifact.
4. Verify: check sources, safety, accessibility, and alignment.
5. Teach: use it in a real setting.
6. Archive: save the artifact and evidence.
7. Improve: revise from what happened.

## What to verify

Every step should leave enough evidence that future you can understand the decision.

## Common mistakes

- Prompting before sourcing.
- Building too big before teaching once.
- Improving without archiving the previous version.
