# Playbook: Use Codex without overwriting content

## The real problem

Agent work can be useful and risky at the same time, especially around authored course text.

## Quick answer

Use the canonical guide: [Use Codex without overwriting content](/kb/ai-coding-agents/use-codex-without-overwriting-content).

## When to use this

Use this before asking an agent to edit lessons, route structure, templates, or public content.

## Workflow

Name the exact task, protect authored content, require a diff, run lint/build, and review every touched file.

## What to verify

No course bodies were rewritten unless explicitly requested.

## Common mistakes

- Giving a broad “make it better” prompt.
- Skipping the diff review.
