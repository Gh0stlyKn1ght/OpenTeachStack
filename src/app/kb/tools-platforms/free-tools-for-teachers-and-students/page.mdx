# Free tools for teachers and students

## The real problem

Free tools can still cost attention, privacy, support time, or student confusion.

## Quick answer

Evaluate purpose, privacy, access, exports, accessibility, and maintenance before adopting a tool.

## When to use this

Use this before adding a tool to a lesson, course site, assignment, or student workflow.

## Workflow

1. Define the teaching job the tool must do.
2. Check account requirements.
3. Check privacy and data handling.
4. Test export or backup options.
5. Confirm accessibility basics.
6. Create a fallback plan.

## What to verify

Check district policy, student age requirements, terms, data collection, and whether students can access it.

## Common mistakes

- Choosing novelty over workflow fit.
- Assuming free means safe.
- Forgetting what happens when the tool changes.
