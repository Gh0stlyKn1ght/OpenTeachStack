# Tools & Platforms

Tools change faster than courses.

This section is the living reference layer for software, documentation, install links, setup notes, and tool comparisons that support OpenTeachStack courses.

## Use this section when

- You do not know what a tool is.
- You need the official install or documentation page.
- You are deciding whether a browser tool is enough or a local development machine is needed.
- You need to compare AI assistants, coding agents, IDEs, terminals, GitHub, hosting, or MCP-connected workflows.
- You want the current link without bloating a lesson with software references.
