# Teachable Export Support

This folder stores instructor-facing packaging metadata for Teachable.

Course folders are intentionally minimal and export-oriented. They should remain in sync with course metadata and lesson outcomes.

Keep each file practical:
- course-description.md
- lesson-outline.csv
- downloads-index.md
- quiz-plan.md
- capstone-or-final-project.md
- video-script-notes.md
- instructor-talking-points.md

Do not mirror lesson content; use this as a curriculum delivery spine.
