# Apps Script for Teacher Automation

## Audience
Teachers adding reliable automation using Apps Script

## Promise
Automate one low-risk workflow with logs, permission checks, and rollback.

## Prerequisites
Google Workspace access and familiarity with Sheets/Docs/Drive.

## Expected Learner Outcomes
Build and test at least one script that creates or updates classroom workflow artifacts from fake data.

## Artifacts Produced
- Automation script, Logger test log, permission matrix, rollback procedure, fake-data test sheet.

## Final Project
One safe automation toolkit with evidence of tested success and a fix path.

## Safety / Privacy / Accessibility
Run tests only on sandbox folders and fake rows. Document permission boundaries and stop procedures.

## Assessment and Review
Sandbox test log, one error example, and teacher review note before classroom use.
