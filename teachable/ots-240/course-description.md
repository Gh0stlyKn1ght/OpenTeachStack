# Open Resources and GitHub for Educators

## Audience
Teachers sharing and publishing open classroom resources

## Promise
Publish reusable resources with clear source, licensing, and maintenance ownership.

## Prerequisites
Basic GitHub workflow for repositories, README, and attribution discipline.

## Expected Learner Outcomes
Create a complete open-resource repo package with source notes, license notes, and exclusion policy.

## Artifacts Produced
- README, source-notes, license-notes, attribution matrix, excluded-media note.

## Final Project
A publish-ready resource repository with review trail and maintenance owner.

## Safety / Privacy / Accessibility
Do not include private student data, internal policy docs, or unvetted third-party material.

## Assessment and Review
Pre-release compliance check for licensing, source traceability, and contributor safety notes.
