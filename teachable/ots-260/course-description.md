# AI Media and Lesson Delivery

## Audience
Teachers improving instructional media quality

## Promise
Build accessible lesson media that improves learning without adding noise.

## Prerequisites
One working lesson and ability to create charts, slides, or short video snippets.

## Expected Learner Outcomes
Create a media packet with alt text, captions, transcript, readability audit, and review points.

## Artifacts Produced
- Media packet, image prompts, diagram plan, accessibility review sheet, revision log.

## Final Project
A classroom-ready media bundle with accessibility and quality criteria ready for reuse.

## Safety / Privacy / Accessibility
Respect copyright, licensing, and student privacy in all media examples.

## Assessment and Review
Rubric-based media review for accessibility, caption quality, and reuse clarity.
