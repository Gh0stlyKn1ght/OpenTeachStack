# Cyber Safety for Educators

## Audience
Teachers strengthening cyber safety routines

## Promise
Use fictionalized scenarios to build a practical safety and incident response packet.

## Prerequisites
No security tools required; focus on policy-aligned workflows.

## Expected Learner Outcomes
Create safety review routines with privacy-safe examples and escalation map.

## Artifacts Produced
- MFA/account hygiene checklist, phishing response template, website safety review, escalation notes.

## Final Project
A practical safety packet with clear escalation, documentation, and maintenance triggers.

## Safety / Privacy / Accessibility
Use redacted, fictional examples only. Never expose real incident details outside allowed channels.

## Assessment and Review
Scenario role-play review and one documented escalation test.
