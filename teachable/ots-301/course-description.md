# Teacher Course Sites

## Audience
Teachers publishing and maintaining course sites

## Promise
Ship an accessible, maintainable course hub with clear public/private boundaries.

## Prerequisites
Basic web publishing familiarity; no deep dev skills required.

## Expected Learner Outcomes
Create a sitemap, map public/private areas, and define maintenance routines for releases.

## Artifacts Produced
- Course hub sitemap, broken-link review, accessibility checklist, maintenance log.

## Final Project
A published course site that is coherent for students and maintainable by a small team.

## Safety / Privacy / Accessibility
Audit links and assets for student-identifying content before public release.

## Assessment and Review
Monthly maintenance review and periodic accessibility check before each publication.
