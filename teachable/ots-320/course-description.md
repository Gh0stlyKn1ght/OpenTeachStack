# AI Coding Agents for Educators

## Audience
Teachers using coding agents for teacher workflow tasks

## Promise
Use AI coding agents safely with scoped prompts, file boundaries, diff review, and rollback.

## Prerequisites
Basic command line and git concepts, ability to review diffs.

## Expected Learner Outcomes
Create a bounded prompt set, diff review notes, and fake-data verification for agent work.

## Artifacts Produced
- Safe prompts, allowed/blocked file list, diff review notes, fake-data test output, rollback note.

## Final Project
An agent-assisted toolkit and workflow that preserves teacher authorship control.

## Safety / Privacy / Accessibility
Never allow broad write access. Keep student work and secrets out of prompts.

## Assessment and Review
Pre-release agent checklist and post-edit reviewer sign-off.
